# PDF导出Agent（PDF Export Agent）

## 一、职责描述

PDF导出Agent负责将所有Agent的分析结果整合渲染为格式化的PDF诊断报告，包括：
1. 生成报告封面（含原图缩略图和报告元数据）
2. 渲染原图分析区（原图展示 + EXIF + 直方图）
3. 渲染构图分析区（九宫格标注图 + 构图评价）
4. 渲染大师对标区（匹配结果表 + 风格对比）
5. 渲染修图教程区（分步参数 + 双平台参数表）
6. 渲染综合总结区（评分 + 优势/不足 + 改进优先级 + 学习建议）
7. 管理标注图和问题图库的引用

## 二、输入规范

```json
{
    "image_path": "用户照片路径",
    "analysis_results": {
        "summary": { "..." },
        "specialist": { "..." },
        "composition": { "..." },
        "style_match": { "..." },
        "retouch": { "..." }
    },
    "subject_type": "landscape",
    "output_dir": "output/",
    "annotated_image_path": "标注后的构图分析图（可选）"
}
```

## 三、处理逻辑

```
步骤1：准备输出环境
  - 确保输出目录存在
  - 生成唯一文件名（时间戳）
  - 初始化ReportLab文档（A4页面）

步骤2：构建封面
  - 标题："摄影诊断报告"
  - 副标题："PHOTOGRAPHY DIAGNOSIS REPORT"
  - 原图缩略图（居中，最大120mm宽）
  - 报告编号和生成日期
  - 深灰色背景 + 暖金色强调

步骤3：构建目录页
  - 五个章节的标题列表

步骤4：构建第一章（原图分析）
  - 1.1 原图展示（居中大图）
  - 1.2 拍摄参数表格
  - 1.3 图像信息（尺寸、文件大小）

步骤5：构建第二章（构图分析）
  - 2.1 三分法分析（匹配度 + 交叉点）
  - 2.2 水平线分析（角度 + 校正建议）
  - 2.3 引导线分析（数量 + 类型）
  - 2.4 构图评分（大字号展示）
  - 2.5 构图改进建议

步骤6：构建第三章（大师对标）
  - 3.1 风格匹配结果表格（排名/大师/匹配度/等级）
  - 3.2 混合风格检测（如有）
  - 3.3 风格发展建议

步骤7：构建第四章（修图教程）
  - 4.1 诊断问题列表（按严重程度着色）
  - 4.2 修图步骤（编号 + 工具 + 参数 + 说明）
  - 4.3 Lightroom参数汇总表格
  - 4.4 醒图参数汇总表格
  - 4.5 预估效果

步骤8：构建第五章（综合总结）
  - 5.1 综合评分（大字号）
  - 5.2 优势列表（绿色标记）
  - 5.3 不足列表（橙色标记）
  - 5.4 改进优先级排序
  - 5.5 学习建议和练习计划

步骤9：PDF渲染
  - 使用ReportLab渲染所有内容
  - 设置页眉页脚
  - 生成PDF文件

步骤10：质量检查
  - 检查文件大小（<10MB）
  - 检查页面完整性
  - 返回PDF文件路径
```

## 四、输出格式

```json
{
    "agent_type": "pdf_export",
    "status": "success",
    "pdf_path": "output/photo_report_20260713_143000.pdf",
    "file_size_mb": 3.5,
    "page_count": 8,
    "generation_time_seconds": 2.3,
    "metadata": {
        "report_id": "PD-20260713-001",
        "generated_at": "2026-07-13T14:30:00+08:00",
        "subject_type": "landscape",
        "overall_score": 72.5
    }
}
```

## 五、PDF设计规范

```
页面尺寸：A4 (210mm x 297mm)
页边距：上25mm 下25mm 左20mm 右20mm
字体：
  - 标题：SimHei Bold 18pt
  - 章节标题：SimHei 16pt
  - 子标题：SimHei 12pt
  - 正文：SimSun Regular 10pt
  - 评分数字：SimHei 24pt
  - 表格内容：SimSun 9pt
配色：
  - 主色：#2C2C2C（深灰）
  - 强调色：#C8A96E（暖金）
  - 正文色：#333333
  - 辅助色：#888888
  - 问题-严重：#CC3333
  - 问题-中等：#CC8833
  - 问题-轻微：#CCAA33
  - 良好：#33AA55
页眉：报告标题（左对齐，浅灰色，8pt）
页脚：页码（右对齐） + "Photo Agent 生成"
```

## 六、Prompt模板

```
你是PDF报告排版专家。

请将以下分析结果整理为格式规范的PDF报告：

## 报告内容
- 原图路径：{image_path}
- 题材类型：{subject_type}
- 综合评分：{overall_score}

## 各章节内容
1. 原图分析：{original_section}
2. 构图分析：{composition_section}
3. 大师对标：{style_section}
4. 修图教程：{retouch_section}
5. 综合总结：{summary_section}

## 设计要求
- 使用深灰+暖金配色方案
- 表格使用交替行底色
- 问题列表按严重程度着色
- 评分使用大字号暖金色展示
- 每章使用分页符分隔
```
