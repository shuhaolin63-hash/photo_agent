# 修图建议Agent（Retouch Agent）

## 一、职责描述

修图建议Agent负责分析照片的技术问题并生成个性化的修图方案，包括：
1. 全面的技术问题诊断（曝光/构图/色彩/对焦/噪点）
2. 生成Lightroom精确修图参数
3. 生成醒图（XingTu）手机端修图参数
4. 生成分步修图教程
5. 修图前后效果预估
6. 修图伦理提示

## 二、输入规范

```json
{
    "image_path": "照片路径",
    "subject_type": "landscape",
    "exif_data": {
        "camera": "Sony A7R IV",
        "lens": "FE 16-35mm f/2.8 GM",
        "focal_length": 24,
        "aperture": 11,
        "shutter_speed": "1/125",
        "iso": 100
    }
}
```

## 三、处理逻辑

```
步骤1：图像读取与预处理
  - 读取图像为OpenCV格式
  - 获取图像尺寸和通道信息
  - 提取EXIF数据（如未提供）

步骤2：全面问题诊断
  - 曝光问题：直方图分析（过曝/欠曝/高光溢出/暗部缺失）
  - 构图问题：水平线倾斜/透视畸变
  - 色彩问题：白平衡偏色/过饱和/不饱和/噪点
  - 对焦问题：全局模糊/局部失焦/运动模糊

步骤3：修图参数生成
  - 基础参数：根据题材选择基础预设
  - 曝光校正：根据曝光问题生成补偿参数
  - 色彩校正：根据偏色问题生成色温/色调调整
  - 清晰度：根据题材和问题生成Clarity/Dehaze
  - 降噪/锐化：根据噪点水平和锐度问题生成参数
  - 变形校正：根据水平线/透视问题生成Transform参数

步骤4：双平台参数输出
  - Lightroom参数：曝光/对比度/高光/阴影/白色/黑色/清晰度/鲜艳度/饱和度/色温/色调/曲线/降噪/锐化/Transform
  - 醒图参数：光感/亮度/对比度/高光/阴影/清晰度/鲜明度/饱和度/色温/色调/锐化/降噪

步骤5：修图步骤序列化
  - 按标准修图流程排序：
    1. 基础校正（水平线/畸变/色差）
    2. 曝光与影调
    3. 色彩调整
    4. 局部精修
    5. 降噪与锐化
    6. 风格增强（可选）

步骤6：修图伦理检查
  - 检查修图建议是否符合伦理
  - 如为纪实照片，标注不允许的修图操作
  - 如为艺术照片，标注自由度范围

步骤7：效果预估
  - 根据问题数量和严重程度预估修图效果
  - 评估修图难度（简单/中等/较难）
```

## 五、Prompt模板

```
你是摄影修图专家。

请分析以下照片的技术问题，并给出修图建议：

## 照片信息
- 题材：{subject_type}
- 拍摄参数：{exif_summary}

## 已诊断问题
{problem_list}

## 要求
1. 给出Lightroom精确修图参数（曝光/对比度/高光/阴影/清晰度/饱和度/色温/降噪/锐化）
2. 同时给出醒图（手机端）对应参数
3. 按标准修图流程（基础校正→曝光影调→色彩→局部→降噪锐化）生成步骤
4. 预估修图后效果和难度
5. 如为纪实照片，标注修图伦理限制

请给出专业、可操作的修图方案。
```
