## 一、技能概述

本技能是**摄影诊断AI Agent系统**的核心知识入口，提供全品类摄影作品的智能诊断能力。系统采用**多Agent协作架构**，结合大语言模型（GPT-4o/通义千问）的语义理解、CLIP的视觉语义匹配、SAM的精确图像分割、OpenCV的量化图像分析能力，为用户提供专业级的摄影诊断服务。

### 核心能力

| 能力维度 | 说明 |
|---------|------|
| **构图诊断** | 三分法、黄金分割、引导线、视觉重心、对称性、前景中景远景层次分析 |
| **光影分析** | 直方图评估、高光溢出检测、阴影死黑检测、曝光均匀度、光线质量评分 |
| **色彩评估** | 色温偏移、色彩饱和度、色彩和谐度、肤色准确度（人像）、情绪氛围匹配 |
| **风格对标** | 将用户作品与对应品类大师作品进行多维风格对比，输出差距分析与改进路径 |
| **修图建议** | 基于诊断结果的参数化修图方案，支持Lightroom/Photoshop/Capture One等工具 |
| **PDF报告** | 将全部诊断结果排版为带图像标注、直方图、对标对比的专业PDF诊断报告 |

### 支持的摄影品类

- **人像摄影** - 肖像/写真/证件照/情侣照/家庭照/婚礼摄影
- **风光摄影** - 山川/海景/日出日落/城市风光/建筑/瀑布/森林
- **纪实摄影** - 街拍/新闻/人文/旅行/美食/节庆/运动
- **星空摄影** - 银河/星轨/极光/流星/天文摄影/夜景
- **静物摄影** - 微距/美食摄影/产品摄影/花卉/珠宝/手表

## 二、触发条件

### 2.1 直接触发（用户明确请求摄影诊断）

当用户输入包含以下意图关键词时，**直接触发**本技能：

| 意图类别 | 触发关键词 |
|---------|-----------|
| 摄影诊断 | `帮我分析这张照片`、`摄影诊断`、`照片点评`、`摄影点评`、`作品分析`、`照片分析` |

| 光影分析 | `光线分析`、`曝光怎么样`、`光影诊断`、`高光溢出`、`欠曝`、`过曝` |
| 色彩评估 | `色彩分析`、`色调评估`、`颜色搭配`、`色温`、`色彩和谐` |
| 风格对标 | `风格对标`、`大师风格`、`像不像某某风格`、`这个风格属于什么流派` |

| 综合诊断 | `帮我看看这张照片`、`这张照片拍得怎么样`、`摄影师来看看`、`照片有没有问题` |

### 2.2 间接触发（用户上传照片并含分析意图）

当用户**上传了图片文件**（.jpg/.jpeg/.png/.webp/.heic/.tiff），且输入文本中包含以下暗示性关键词时，**间接触发**本技能：

| 暗示类别 | 触发关键词 |
|---------|-----------|
| 质量询问 | `拍得好不好`、`水平怎么样`、`还能不能更好`、`有没有进步` |
| 技术讨论 | `快门速度`、`光圈`、`ISO`、`焦段`、`这个参数对不对`、`用什么镜头拍的` |
| 比较请求 | `和大师比怎么样`、`这水平能发朋友圈吗`、 `达到专业水平了吗` |
| 学习意图 | `怎么提高`、 `有什么建议`、 `我该怎么练`、 `想学摄影` |
| 后期讨论 | `色调怎么调`、`加个滤镜`、`这个颜色不对`、`偏色了` |

| 星空 | `星星`、`银河`、`星轨`、`极光`、`夜景`、`月亮`、`流星`、`天体` |

## 四、四层诊断架构

本系统采用**四层深度诊断架构**，逐层递进、由表及里地对照片进行全方位诊断：

### 4.1 第一层：分析层（Analytical Layer）

> 目标：客观量化照片的技术参数与视觉表现

| 分析维度 | 使用技术 | 输出指标 |
|---------|---------|---------|
| **构图分析** | OpenCV + LLM | 三分法合规度、黄金分割位置、引导线方向、视觉重心坐标、对称性评分、留白比例 |
| **光影分析** | OpenCV直方图 + LLM | 曝光偏差值、高光溢出占比（%）、阴影死黑占比（%）、动态范围利用、光线方向与质量 |
| **色彩分析** | OpenCV HSV + LLM | 色温偏差值、主色调提取、色彩饱和度分布、色彩和谐度评分、肤色偏差（人像） |
| **清晰度分析** | OpenCV Laplacian + LLM | 整体锐度评分、边缘清晰度、噪点水平（SNR）、运动模糊检测、景深范围评估 |

### 4.2 第二层：对标层（Benchmarking Layer）

> 目标：将分析结果与大师作品进行风格维度的系统对比

对标流程：
1. **CLIP语义匹配** - 从大师风格库中检索与当前照片语义最相似的3-5位大师作品
2. **风格维度提取** - 在5个维度上对用户照片和大师作品进行量化对比：
   - 用光风格（光质/光位/光比）
   - 构图模式（构图法/打破规则程度）
   - 色调倾向（色温/饱和度/对比度）
   - 叙事强度（情感传达/故事性/氛围感）
   - 后期风格（自然/胶片/数码/混合）
3. **差距可视化** - 生成风格雷达图，直观展示差距与优势
4. **改进路径** - 基于差距分析生成优先级排序的改进建议

### 4.3 第三层：修图层（Retouching Layer）

> 目标：根据诊断结果生成可操作的修图方案

分场景修图方案：

| 场景 | 修图重点 | 典型调整参数 |
|------|---------|-------------|
| **人像** | 肤色统一、瑕疵修复、眼神光、背景虚化、液化塑形 | 肤色色温偏移 ±200K，清晰度+15，柔和皮肤，HSL肤色微调 |
| **风光** | 动态范围恢复、色彩层次、透视校正、全景拼接 | 高光-40，阴影+60，饱和度+10，清晰度+20，渐变滤镜 |
| **纪实** | 黑白转换、对比度增强、裁剪强化、色彩分离 | 明度混合器调整，对比度+25，高反差黑白，局部提亮 |
| **星空** | 噪声抑制、星点锐化、银河色彩还原、前景融合 | ISO噪声 reduction，色温3500K→4500K，HSL蓝紫色调整 |
| **静物** | 白平衡校正、产品色彩准确、质感增强、背景处理 | 白平衡精调，清晰度+30，局部对比度增强，曝光补偿 |

### 4.4 第四层：输出层（Output Layer）

> 目标：将全部诊断结果结构化输出为专业PDF报告

输出报告结构（7个核心章节）：

```
第1章 - 照片概览
    ├── 原图展示（带EXIF信息覆盖层）
    └── 基础参数摘要表

第2章 - 构图诊断
    ├── 构图辅助线叠加分析图
    ├── 三分法/黄金分割合规度
    ├── 视觉重心标注
    └── 构图综合评分（0-100分）

第3章 - 光影分析
    ├── RGB/亮度直方图
    ├── 高光/阴影区域热力图
    ├── 曝光评分与建议
    └── 光线质量评估

第4章 - 色彩评估
    ├── 色彩分布饼图
    ├── 色温/色调分析
    ├── 色彩和谐度评分
    └── 肤色准确度（人像场景）

第5章 - 风格对标
    ├── 匹配大师介绍与代表作品
    ├── 风格维度雷达图
    ├── 差距分析详解
    └── 改进优先级排序

第6章 - 修图建议
    ├── 全局调整参数表
    ├── 局部精修方案
    ├── 推荐预设
    └── 后期工作流步骤图

第7章 - 总结与提升路径
    ├── 综合评分（多维度加权）
    ├── 核心改进方向TOP3
    ├── 推荐学习资源
    └── 针对性练习建议
```

## 五、使用方式

### 5.1 标准诊断流程

```
用户上传照片
    │
    ▼
[Step 1] 场景自动识别（CLIP + 关键词辅助）
    │       ├─ 人像 ──→ portrait_agent
    │       ├─ 风光 ──→ landscape_agent
    │       ├─ 纪实 ──→ documentary_agent
    │       ├─ 星空 ──→ starry_agent
    │       └─ 静物 ──→ still_life_agent
    │
    ▼
[Step 2] 专业诊断（场景Agent + OpenCV量化分析）
    │       ├─ 构图分析
    │       ├─ 光影分析
    │       ├─ 色彩分析
    │       └─ 清晰度分析
    │
    ▼
[Step 3] 风格对标（style_agent + CLIP匹配）
    │       ├─ 大师检索
    │       ├─ 维度对比
    │       └─ 差距分析
    │
    ▼
[Step 4] 修图建议（retouch_agent + 场景化方案）
    │       ├─ 全局调整参数
    │       ├─ 局部精修方案
    │       └── 工具适配
    │
    ▼
[Step 5] PDF报告生成（pdf_export_agent）
    │       ├─ 图像标注渲染
    │       ├─ 报告排版
    │       └── PDF输出
    │
    ▼
交付给用户
```

### 5.2 快速诊断模式

用户可指定仅执行部分诊断步骤：

- `只做构图分析` → 仅执行Step 2的构图分析
- `帮我对标大师风格` → 仅执行Step 3的风格对标
- `给我修图参数` → 执行Step 2 + Step 4
- `完整诊断` → 执行全部Step 1-5

### 5.3 学习路径建议

本系统不仅提供诊断，还提供摄影能力提升路径：

```
初学者路径：
    构图基础（三分法→引导线→负空间）
    → 光线入门（自然光→黄金时段→基础布光）
    → 色彩基础（色温→白平衡→色彩和谐）
    → 后期入门（基础调整→预设使用→RAW处理）

进阶者路径：
    构图进阶（打破规则→视觉叙事→情绪构图）
    → 光线进阶（混合光→人造光→创意用光）
    → 色彩进阶（色彩分级→情绪调色→风格化）
    → 后期进阶（分层精修→蒙版运用→工作流优化）

专业者路径：
    风格建立（大师研究→个人风格→作品集策划）
    → 技术精进（极端场景→器材极限→后期工业化）
    → 大师对标（精准对标→超越对标→自成一派）
```

## 七、注意事项

1. **隐私安全** - 系统自动检测照片中的个人信息（人脸/车牌等），支持PII脱敏处理
2. **模型切换** - GPT-4o为首选LLM，通义千问为备用fallback，自动切换
3. **性能限制** - 建议单日诊断不超过20张照片，单张照片分析耗时约60-120秒
4. **评分标准** - 所有评分采用0-100分制，参考专业摄影比赛评审标准
5. **风格对标** - 对标结果仅供学习参考，不构成创作指导或侵权建议
6. **离线模式** - CLIP和SAM支持本地离线推理，仅LLM部分需要联网
