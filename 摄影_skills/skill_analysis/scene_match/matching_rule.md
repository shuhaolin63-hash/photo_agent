# 场景匹配规则

## 一、概述

场景匹配（Scene Matching）是指根据照片的内容特征，自动或半自动地判断照片属于哪种摄影类型（人像/风光/纪实/星空/静物等），从而为后续的参数推荐、后期调色、构图分析提供上下文依据。

在摄影AI Agent系统中，场景匹配是第一步决策节点——**正确分类场景是一切后续建议的基础**。

---

## 二、场景分类体系

### 2.1 一级分类（7大类）

| 编号 | 场景类别 | 英文标识 | 核心特征 |
|------|----------|---------|----------|
| S01 | 人像 | portrait | 人物为主要主体，占画面面积 > 15% |
| S02 | 风光 | landscape | 自然/城市景观为主体，无突出人物 |
| S03 | 纪实/街拍 | documentary | 捕捉生活瞬间，有人物但非主体 |
| S04 | 星空/夜景 | nightscape | 夜间场景，含星空/月亮或城市夜景灯光 |
| S05 | 静物/美食 | still_life | 小型物体特写，通常在可控光线下拍摄 |
| S06 | 建筑 | architecture | 建筑物为主体，强调结构/几何/空间 |
| S07 | 运动/体育 | sports | 运动中的主体，强调速度/力量/动态 |

### 2.2 二级细分

#### S01 人像细分

| 子类 | 特征 | 场景标签 |
|------|------|----------|
| 室内影棚人像 | 纯色/渐变背景，可控布光 | portrait.studio |
| 户外自然光人像 | 自然环境背景，自然光/反光板 | portrait.outdoor |
| 环境人像 | 人物与环境互动，环境讲述故事 | portrait.environmental |
| 特写/面部 | 面部占画面 50%+，强调表情 | portrait.closeup |
| 全身/时尚 | 全身构图，强调服装/姿态 | portrait.fashion |
| 情侣/婚纱 | 两人互动，浪漫氛围 | portrait.couple |
| 群体合影 | 多人排列，需要均匀照明 | portrait.group |

#### S02 风光细分

| 子类 | 特征 | 场景标签 |
|------|------|----------|
| 山景 | 山脉/山峰为主体 | landscape.mountain |
| 海景/海岸 | 海洋/海岸线为主体 | landscape.sea |
| 森林/树木 | 树木/林间场景 | landscape.forest |
| 日出/日落 | 天空色彩戏剧性变化 | landscape.sunrise_sunset |
| 极光/特殊天象 | 极光、彩虹、闪电等 | landscape.phenomenon |
| 城市风光 | 城市天际线、全景 | landscape.urban |
| 瀑布/河流 | 水体特写 | landscape.water |

#### S04 星空/夜景细分

| 子类 | 特征 | 场景标签 |
|------|------|----------|
| 银河/星空 | 星空为主体，无地面/少量地面前景 | nightscape.milky_way |
| 星轨 | 长曝光星轨效果 | nightscape.star_trail |
| 月亮特写 | 月亮为主体 | nightscape.moon |
| 城市夜景 | 城市灯光、车流光轨 | nightscape.city |
| 蓝调时刻 | 日落后蓝调天空 + 城市灯光 | nightscape.blue_hour |
| 烟花/灯光秀 | 特殊夜间事件 | nightscape.fireworks |

## 三、CLIP分类逻辑

### 3.1 CLIP模型概述

CLIP（Contrastive Language-Image Pre-training）是OpenAI提出的多模态模型，能够在图像和文本之间建立语义关联。在摄影场景匹配中，CLIP可以：

1. **零样本分类**：无需训练，直接用文本描述匹配图像类别
2. **灵活扩展**：随时添加新的场景类别描述
3. **细粒度匹配**：不仅判断类别，还能给出各类别的置信度

### 3.2 场景匹配的Prompt设计

为每个场景类别设计高质量的文本描述（prompt），是CLIP分类效果的关键：

#### 人像场景Prompt模板

```
"一张人像照片，人物面部清晰可见，[室内/户外]环境，[柔和/戏剧性]光线，
[正面/侧面]角度，[特写/半身/全身]构图，[自然/影棚]布光"
```

#### 风光场景Prompt模板

```
"一张风光摄影作品，[山脉/海洋/森林/城市]景观，[日出/正午/日落/蓝调时刻]光线，
[广阔/细节]视角，[晴朗/多云/雾气]天气，[壮观/宁静/戏剧性]氛围"
```

#### 星空场景Prompt模板

```
"一张星空摄影作品，[银河/星轨/月亮/城市夜景]，[深邃/绚丽/寂静]夜空，
[地面/无地面]前景，[长曝光/单次曝光]效果"
```

### 3.3 分类决策流程

```
输入图像
    │
    ▼
CLIP特征提取
    │
    ▼
并行计算各类别相似度得分
    │
    ▼
┌─────────────────┐
│ 得分 > 0.25？    │──否──→ "不确定/混合场景" → 人工判定
│                      │
│ 是                    │
│ ▼                     │
│ Top-1 类别与         │
│ Top-2 得分差 > 0.1？ │──否──→ "混合场景" → 取两者标签
│                      │
│ 是                    │
│ ▼                     │
│ 输出 Top-1 类别      │
│ + 置信度             │
└─────────────────┘
    │
    ▼
二级细分分类（同一逻辑递归）
    │
    ▼
输出完整场景标签
    │
    ▼
路由到对应的参数推荐/调色方案
```

### 3.4 置信度阈值设定

| 置信度 | 决策 | 处理方式 |
|--------|------|----------|
| ≥ 0.70 | 高置信度 | 直接使用该分类，应用对应参数方案 |
| 0.50 - 0.69 | 中等置信度 | 使用该分类，但标注"建议人工确认" |
| 0.25 - 0.49 | 低置信度 | 列出前 3 候选类别，建议用户选择 |
| < 0.25 | 极低置信度 | 无法确定，提示用户手动指定场景 |

## 四、多模态辅助判断

### 4.1 结合EXIF数据的判断

除了CLIP视觉分类，还可以利用照片的EXIF元数据辅助判断：

| EXIF字段 | 判断规则 | 权重 |
|----------|----------|------|
| **快门速度 > 10s** | 高概率为星空/夜景/车流光轨 | 强 |
| **快门速度 < 1/1000s** | 高概率为运动/体育/飞鸟 | 强 |
| **光圈 < f/2.8** | 高概率为人像/低光 | 中 |
| **焦距 > 200mm** | 高概率为运动/野生动物/月亮 | 中 |
| **焦距 < 24mm** | 高概率为风光/建筑/星空 | 中 |
| **ISO > 3200** | 高概率为星空/室内/夜间 | 中 |
| **GPS坐标** | 城市区域→偏建筑/城市风光；自然保护区→偏风光 | 弱 |

### 4.2 结合图像分析的判断

| 特征 | 检测方法 | 场景推断 |
|------|----------|----------|
| **肤色检测** | HSV色彩空间中肤色范围匹配 | 存在大量肤色像素 → 人像概率高 |
| **星空检测** | 暗区域中高频亮点的密度统计 | 密集亮点 → 星空 |
| **水平线检测** | 边缘检测 + 霍夫变换 | 明确水平线 → 风光/海景 |
| **色彩分布** | 直方图分析 | 暖色为主 → 日出日落；蓝色为主 → 夜景/蓝调 |
| **模糊区域** | 局部拉普拉斯方差 | 背景模糊 → 人像（浅景深）；全模糊 → 运动 |

### 4.3 综合决策

```
最终场景 = w1 × CLIP分类 + w2 × EXIF推断 + w3 × 图像特征推断

其中：
w1 = 0.60（视觉分类权重最高）
w2 = 0.20（EXIF辅助）
w3 = 0.20（图像特征辅助）
```

## 五、场景匹配后的参数推荐

### 5.1 参数映射表

| 场景类别 | 推荐光圈 | 推荐快门 | 推荐ISO | 推荐白平衡 | 推荐构图法 |
|----------|----------|----------|---------|------------|------------|
| 人像特写 | f/1.4-f/2.8 | 1/125s+ | 100-400 | 5200-5800K | 三分法/中心 |
| 户外人像 | f/2-f/4 | 1/250s+ | 100-200 | 5200K(AWB) | 三分法/引导线 |
| 风光广角 | f/8-f/11 | 三脚架不限 | 100 | 日光(5500K) | 三分法/引导线 |
| 山脉风光 | f/8-f/11 | 1/250s+ | 100-400 | 日光(5500K) | 三分法/层次 |
| 星空 | f/1.4-f/2.8 | 20-25s | 3200-6400 | 3800-4500K | 中心/前景引导 |
| 城市夜景 | f/8-f/11 | 10-30s | 100-200 | 4500-5500K | 三分法/对称 |
| 静物 | f/5.6-f/11 | 三脚架不限 | 100 | 自定义 | 三分法/框架 |
| 建筑外景 | f/8-f/11 | 1/125s+ | 100-200 | 日光(AWB) | 对称/引导线 |
| 运动体育 | f/2.8-f/5.6 | 1/500s+ | 自动 | AWB | 三分法/动态 |
| 纪实街拍 | f/5.6-f/8 | 1/250s+ | 自动 | AWB | 三分法/框架 |

### 5.2 后期调色方案映射

| 场景类别 | 推荐调色方向 | HSL重点调整 |
|----------|-------------|-------------|
| 人像 | 自然肤色、柔和高光 | 橙色饱和度微调、红色明度提升 |
| 风光 | 强化天空和植被色彩 | 蓝色饱和度+、绿色饱和度+、青色明度- |
| 星空 | 深蓝夜空、暖色星点 | 蓝色饱和度+、蓝色明度-、紫色微调 |
| 纪实 | 真实还原、不过度调色 | 微调或保持原色 |
| 静物/美食 | 暖色调、食欲感 | 黄色/橙色饱和度+、明度+ |
| 建筑 | 中性偏冷、线条清晰 | 蓝色/青色微调、饱和度适中 |
| 运动 | 高对比、充满力量 | 对比度+、饱和度+、蓝色增强 |
