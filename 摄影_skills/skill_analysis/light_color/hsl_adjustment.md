# HSL调色理论

## 一、HSL色彩模型基础

### 1.1 什么是HSL

HSL（Hue, Saturation, Lightness）是摄影后期调色中最直观、最常用的色彩模型：

- **H（Hue/色相）**：颜色的基本属性，即"是什么颜色"。色相用0-360度表示，对应色轮上的角度
- **S（Saturation/饱和度）**：颜色的纯度或鲜艳程度。0%为灰，100%为纯色
- **L（Lightness/明度/亮度）**：颜色的明暗程度。0%为纯黑，100%为纯白

### 1.2 色轮与色相对应关系

```
         0°/360°
          红色
           |
  330° 品红 ——— 30° 橙色
       |              |
  300° 紫色       60° 黄色
       |              |
  270° 蓝紫 ——— 90° 黄绿
       |              |
  240° 蓝色       120° 绿色
       |              |
  210° 青蓝 ——— 150° 青绿
           |
         180° 青色
```

**相邻色**：色轮上相邻的颜色（如红-橙、蓝-紫），搭配和谐
**互补色**：色轮上相对的颜色（如红-青、蓝-黄），对比强烈
**三色组合**：色轮上等距的三个颜色（如红-绿-蓝），视觉平衡

### 1.3 HSL与RGB的关系

- RGB是加色模型（发光体），HSL是感知模型（更符合人眼对颜色的理解）
- Lightroom/Camera Raw 的"HSL面板"实质上是在RGB色彩空间中操作，但以HSL的直观方式呈现
- 调整HSL不会改变画面的明暗结构（亮度通道），这是它与曲线调色的关键区别

## 二、各通道调整方法

### 2.1 色相（Hue）调整

**功能**：改变某一颜色的基本类别。

**调整方向**：
- 红色通道：向橙/品红方向偏移
- 橙色通道：向红/黄方向偏移
- 黄色通道：向绿/橙方向偏移
- 绿色通道：向青/黄绿方向偏移
- 青色通道：向蓝/绿方向偏移
- 蓝色通道：向紫/青方向偏移
- 紫色通道：向品红/蓝方向偏移
- 品红通道：向紫/红方向偏移

**常见应用**：
| 调整 | 效果 | 用途 |
|------|------|------|
| 绿色向青偏移 | 绿叶变青绿 | 冷调风光 |
| 橙色向红偏移 | 皮肤偏红 | 健康肤色或加强肤色红润 |
| 蓝色向青偏移 | 天空偏青蓝 | 电影感色调 |
| 黄色向橙偏移 | 秋叶更金黄 | 秋色风光 |

**注意事项**：
- 色相调整幅度不宜过大（通常 ±15-20 即可），过度会导致色彩不自然
- 色相滑块影响的颜色范围较宽，可能产生连带影响（如调整红色时可能影响皮肤）
- Lightroom 的"目标调整工具"可以精准选择要调整的颜色

### 2.2 饱和度（Saturation）调整

**功能**：调整某一颜色的纯度/鲜艳程度。

**调整原则**：
- **全局饱和度**：增加全局饱和度会让所有颜色更鲜艳，但也可能让已经鲜艳的颜色"溢出"
- **局部饱和度**：只增强或减弱特定颜色的饱和度，更精确可控

**常见策略**：

| 策略 | 操作 | 效果 |
|------|------|------|
| **突出主体** | 提升主体颜色的饱和度，降低环境色饱和度 | 让主体色彩"跳"出来 |
| **复古色调** | 降低全局饱和度至 -20 至 -40 | 模拟胶片褪色效果 |
| **选择性饱和** | 仅提升蓝色（天空）和绿色（草地）饱和度 | 风光摄影经典手法 |
| **皮肤处理** | 降低橙色/红色饱和度 -10 至 -20 | 避免皮肤过红，更柔和自然 |

**注意**：饱和度过高会导致颜色"溢出"（颜色值超过255），细节丢失且无法恢复。

### 2.3 明度（Luminance）调整

**功能**：调整某一颜色的亮度/明暗。

**与曝光的区别**：曝光调整是全局明暗变化，明度通道是针对特定颜色调整明暗。例如：压暗蓝色明度 = 让画面中的蓝色区域变暗（蓝色天空变深），但不影响红色/绿色区域。

**常见策略**：

| 策略 | 操作 | 效果 |
|------|------|------|
| **压暗天空** | 降低蓝色/青色明度 | 天空变深，戏剧性增强 |
| **提亮皮肤** | 提升橙色明度 | 皮肤更明亮通透 |
| **压暗绿色** | 降低绿色明度 | 暗调森林，神秘感 |
| **提亮黄色** | 提升黄色明度 | 金色阳光/秋叶更明亮 |

## 三、常见调色思路

### 3.1 日系清新色调

**特征**：低饱和度、整体偏亮、略偏青绿

**HSL调整参数参考**：
| 通道 | 色相 | 饱和度 | 明度 |
|------|------|--------|------|
| 红色 | -5 | -20 | +10 |
| 橙色 | 0 | -15 | +15 |
| 黄色 | +10 | -30 | +20 |
| 绿色 | +15 | -10 | +10 |
| 蓝色 | +10 | -15 | +5 |
| 全局 | - | -20 | - |

**辅助调整**：曝光 +0.3 至 +0.7，对比度 -20，色温偏冷（5500-5800K）

### 3.2 电影感色调（Teal & Orange）

**特征**：阴影偏青蓝（Teal），高光偏橙（Orange），形成强烈的互补色对比

**HSL调整参数参考**：
| 通道 | 色相 | 饱和度 | 明度 |
|------|------|--------|------|
| 蓝色 | +20 | +15 | -10 |
| 青色 | +10 | +20 | -5 |
| 橙色 | +5 | +15 | +5 |
| 红色 | -5 | +5 | -5 |
| 绿色 | 0 | -20 | -10 |
| 全局 | - | -10 | - |

**辅助调整**：色温偏暖（4800-5000K），高光偏橙，阴影偏青蓝（色调曲线或分离色调）

### 3.3 复古胶片色调

**特征**：降低全局饱和度，高光偏黄绿，暗部偏蓝紫，有褪色感

**HSL调整参数参考**：
| 通道 | 色相 | 饱和度 | 明度 |
|------|------|--------|------|
| 红色 | -10 | -25 | -5 |
| 橙色 | -5 | -20 | 0 |
| 黄色 | +15 | -15 | +10 |
| 绿色 | +10 | -10 | -5 |
| 蓝色 | -10 | -20 | -10 |

**辅助调整**：添加颗粒效果，适当降低清晰度，高光偏黄绿（分离色调），暗部偏青蓝

### 3.4 高调/低调人像

**高调人像HSL**：
- 整体偏亮，橙色/红色明度 +15 至 +25
- 饱和度适中或略低，皮肤不过饱和
- 蓝色明度提升，避免阴影过暗

**低调人像HSL**：
- 整体偏暗，蓝色/青色明度 -15 至 -25
- 橙色饱和度降低 -10，避免肤色过突出
- 对比色饱和度增强（如蓝色），在暗调中保持色彩层次

### 3.5 黑白转换的HSL思维

转换为黑白后，HSL中的"明度"滑块变为各颜色的亮度混合比例：

| 颜色 | 提升明度 | 降低明度 |
|------|----------|----------|
| 红色 | 皮肤更亮 | 嘴唇更暗 |
| 橙色 | 整体面部更亮 | 面部变暗 |
| 黄色 | 高光更亮 | 高光变暗 |
| 绿色 | 植物变亮 | 植物变暗（剪影效果） |
| 蓝色 | 天空变亮（通常不需要） | 天空变暗（戏剧性） |

**经典组合**：蓝色明度降低 + 红色明度提升 = 戏剧性黑白人像（暗色天空 + 明亮皮肤）

## 四、HSL与其他调色工具的配合

### 4.1 调色工具层级

| 层级 | 工具 | 作用范围 | 精确度 |
|------|------|----------|--------|
| **全局** | 色温/色调、曝光、对比度 | 整个画面 | 低 |
| **区域** | 色调曲线、HSL面板 | 按亮度区间或颜色区间 | 中 |
| **局部** | 渐变滤镜、调整画笔、径向滤镜 | 画面中指定区域 | 高 |
| **遮罩** | 颜色范围遮罩、亮度范围遮罩 | AI识别特定颜色/亮度区域 | 最高 |

### 4.2 推荐调色工作流

1. **基础校正**：曝光、对比度、高光/阴影、白平衡
2. **色调曲线**：建立明暗对比和整体色调倾向（S曲线增强对比度）
3. **HSL调整**：精细控制各颜色的色相、饱和度、明度
4. **分离色调**：高光和阴影分别添加颜色倾向
5. **局部调整**：用画笔/渐变对特定区域做精确调色
6. **锐化与降噪**：最后的细节处理

## 五、常见错误

| 错误 | 表现 | 解决方案 |
|------|------|----------|
| **过度饱和** | 颜色"炸开"、不自然 | 全局饱和度 -10 至 -20，局部饱和度微调 |
| **色相偏移连带影响** | 改变一个颜色，另一个颜色被意外改变 | 使用"目标调整工具"精确点击要调整的区域 |
| **明度调整破坏层次** | 颜色变亮/暗后丢失细节 | 明度调整幅度控制在 ±15 以内 |
| **忽略白平衡** | 色相调整无法补偿错误的色温 | 先校正白平衡，再进行HSL微调 |
| **黑白转换不考虑颜色明度** | 黑白照片中主体与背景融为一体 | 先调整HSL明度使各颜色有足够明暗差异，再转黑白 |
| **忽视色彩和谐** | 多种颜色色相偏移方向冲突 | 参考色轮确保色彩搭配和谐 |
