# 白平衡与色温

## 一、色温基础理论

### 1.1 色温的定义

色温（Color Temperature）是描述光源颜色特性的物理量，单位为开尔文（Kelvin, K）。其概念源于黑体辐射理论——当一个理想黑体被加热到不同温度时，它会发出不同颜色的光。

**核心原理**：色温数值越低，光线越偏暖（黄/橙/红）；色温数值越高，光线越偏冷（蓝/紫）。

### 1.2 色温数值与光源类型对照

| 色温范围（K） | 光线颜色 | 典型光源 | 情绪/氛围 |
|---------------|----------|----------|-----------|
| 1000-2000K | 深橙色/烛光 | 烛光、火柴、壁炉火焰 | 温暖、亲密、原始 |
| 2000-3000K | 橙黄色 | 钨丝灯（60W/100W）、日落前30分钟 | 温馨、怀旧、居家 |
| 3000-4000K | 黄白色 | 卤素灯、日出/日落黄金时段 | 舒适、柔和、浪漫 |
| 4000-5000K | 中性白 | 日光荧光灯、闪光灯、正午阴天 | 自然、清晰、中性 |
| 5000-5500K | 标准日光 | 正午阳光（晴空）、标准日光灯 | 标准、准确、均衡 |
| 5500-6500K | 微蓝白色 | 电子闪光灯、阴天、日光荧光灯 | 冷静、现代、干净 |
| 6500-7500K | 蓝白色 | 阴天户外、背阴处 | 清冷、忧郁、寂静 |
| 7500-10000K | 蓝色 | 蓝调时刻（日落后/日出前天空） | 冷峻、神秘、深沉 |
| 10000K+ | 深蓝色 | 蓝天无云、阴影处（蓝天反射） | 寒冷、科技、超现实 |

### 1.3 微倒度（Mired）

微倒度（Micro Reciprocal Degree, Mired）是色温的另一种表达方式，更适合表示色温调整的幅度：

```
Mired = 1,000,000 / K

例如：
5500K → 182 Mired
3200K → 313 Mired
差值 = 131 Mired
```

**微倒度的优势**：色温的数值间隔是非线性的（从 2000K 到 3000K 的视觉差异与从 6000K 到 7000K 的视觉差异不同），而微倒度是线性的——131 Mired 的变化在任何色温段都对应相同的视觉色偏量。

色温转换滤镜（CTO/CTB）和滤镜系统常用微倒度标定。

## 二、白平衡原理

### 2.1 白平衡的目的

白平衡（White Balance, WB）的核心目的是：**让白色物体在不同光源下都呈现为白色**。相机通过调整红、绿、蓝三个通道的增益比例来实现这一点。

### 2.2 白平衡模式

| 模式 | 色温设定 | 适用场景 |
|------|----------|----------|
| **AWB（自动白平衡）** | 相机自动判断 | 日常拍摄，光线条件稳定 |
| **日光（Daylight）** | 约 5200-5500K | 晴天户外正午 |
| **阴天（Cloudy）** | 约 6000-6500K | 阴天户外，比日光更暖一些 |
| **阴影（Shade）** | 约 7000-7500K | 阴影中，补偿蓝天反射的蓝色调 |
| **钨丝灯（Tungsten）** | 约 3200K | 室内钨丝灯环境 |
| **荧光灯（Fluorescent）** | 约 4000-4500K | 荧光灯照明（有多种变体） |
| **闪光灯（Flash）** | 约 5500K | 使用外接闪光灯 |
| **自定义（Custom）** | 手动测量 | 混合光或需要精确色温控制 |
| **手动设定（Kelvin）** | 摄影师指定具体K值 | 创意白平衡或精确控制 |

### 2.3 自定义白平衡方法

1. **白纸/灰卡法**：在拍摄场景中放置标准白卡或18%灰卡，相机对准它进行自定义白平衡测量
2. **ExpoDisc**：安装在镜头前的漫射工具，将光源信息传递给相机
3. **RAW后期调整**：拍摄RAW格式，在后期软件中精确调节色温滑块
4. **预设色温**：根据已知光源直接输入K值

## 三、创意白平衡

### 3.1 刻意设置"错误"白平衡

将白平衡故意设置为与场景色温不匹配的值，可以创造独特的色彩氛围：

| 拍摄场景 | 光源色温 | 设置的白平衡 | 效果 |
|----------|----------|---------------|------|
| 黄昏户外（约4000K） | 暖色光 | 阴影（7500K）或更高 | 进一步强化暖色，增强"金色时刻"氛围 |
| 蓝调时刻（约8000K） | 冷色光 | 日光（5500K）或钨丝灯（3200K） | 画面变蓝，强化冷冽/忧郁感 |
| 钨丝灯室内（3200K） | 暖色光 | 日光（5500K） | 画面偏蓝，营造夜间/都市感 |
| 阴天户外（6500K） | 冷色光 | 钨丝灯（3200K） | 画面偏蓝，强化寒冷感 |
| 晴天户外（5500K） | 中性光 | 阴影（7500K） | 画面偏暖，营造复古/怀旧感 |

### 3.2 拍摄策略

**前期策略**：在相机中设定色温，通过取景器实时预览创意效果。适合追求"一次成像"的摄影师。

**后期策略**：拍摄RAW格式，保留完整的色温调节空间。RAW文件的色温调整是无损的，可以在后期精确到 ±100K 的精度。适合需要灵活调整的工作流。

**推荐**：除非有明确的艺术意图或特殊需求，**始终拍摄RAW格式**，在后期调整白平衡。JPEG格式的白平衡调整会损失画质。

## 四、混合光源的白平衡处理

### 4.1 混合光场景

现实场景中常常存在多种不同色温的光源同时照明：

- **室内 + 窗户**：钨丝灯（3200K）+ 日光（5500K）
- **舞台灯光**：多种彩色灯光 + 白色灯光
- **城市夜景**：钠灯（2200K橙黄）+ LED广告牌（6500K蓝白）+ 荧光灯（4000K）
- **画廊**：卤素射灯（3000K）+ 天窗日光（5500K）

### 4.2 处理策略

| 策略 | 方法 | 效果 |
|------|------|------|
| **统一到暖色** | 以暖色光源为基准，接受冷色区域偏蓝 | 营造温暖的室内氛围 |
| **统一到冷色** | 以冷色光源为基准，接受暖色区域偏黄 | 营造冷峻的现代感 |
| **分区处理** | 后期中分层调整不同色温区域 | 两种颜色都准确还原 |
| **利用Gel（色片）** | 在闪光灯上加色片匹配环境光色温 | 让闪光灯与环境光统一 |

### 4.3 CTO/CTB 色片系统

| 滤镜类型 | 功能 | 常见规格 |
|----------|------|----------|
| **CTO（Color Temperature Orange）** | 将冷色光变暖 | 1/4 CTO（约 +45 Mired）、1/2 CTO（约 +80 Mired）、Full CTO（约 +130 Mired） |
| **CTB（Color Temperature Blue）** | 将暖色光变冷 | 1/4 CTB、1/2 CTB、Full CTB |
| **Plus Green / Minus Green** | 匹配荧光灯的绿色调偏移 | 用于混合荧光灯环境 |

## 五、不同题材的白平衡策略

### 5.1 人像摄影

- **自然肤色**：白平衡应确保肤色准确还原，皮肤色温约 5500-5800K
- **暖调人像**：稍偏暖的白平衡（4800-5000K），肤色更红润健康
- **冷调人像**：偏冷的白平衡（6500-7000K），营造时尚/高级感
- **影棚**：闪光灯统一色温，设定5500K或使用自定义白平衡

### 5.2 风光摄影

- **日出/日落**：白平衡设为日光（5500K）或更高，保留金色/橙色暖调
- **蓝天**：白平衡设为阴天（6500K），让天空更蓝
- **雾景**：白平衡略偏冷，强化雾的清冷感
- ** Aurora极光**：通常自动白平衡即可，必要时手动微调

### 5.3 纪实/街头摄影

- AWB通常足够应对混合光源
- 特定场景（如市场灯光）可手动设定突出环境色温
- 夜景街头可利用钠灯/LED灯的色温差异创造视觉趣味

### 5.4 星空摄影

- 手动设定色温 3500-4500K 左右
- 太暖会失去天空的深邃蓝色
- 太冷会失去星云的暖色细节
- 后期RAW可精细调整

## 六、色温与情绪

| 色温区间 | 情绪联想 | 文化象征 | 典型题材 |
|----------|----------|----------|----------|
| 2000-3000K | 温暖、亲密、怀旧、安全 | 家、壁炉、记忆 | 家庭、怀旧、浪漫 |
| 3000-4500K | 舒适、放松、自然 | 阳光、午后、宁静 | 日常、人文、美食 |
| 4500-5500K | 中性、客观、真实 | 白天、工作、清晰 | 产品、新闻、纪实 |
| 5500-7500K | 冷静、忧郁、科技 | 雨天、冬夜、孤独 | 都市、科幻、工业 |
| 7500K+ | 神秘、超现实、梦境 | 极地、太空、未知 | 极光、夜空、超现实 |

**色彩心理学要点**：
- 暖色温让人联想到火光和阳光，产生安全感
- 冷色温让人联想到夜空和冰雪，产生疏离感
- 电影和广告中大量运用色温变化来暗示时间、情绪和叙事走向
