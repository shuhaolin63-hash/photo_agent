# 对焦与景深

## 一、对焦基础

### 1.1 对焦的本质

对焦（Focus）是使光线通过镜头后精确汇聚在传感器平面上，从而获得清晰影像的过程。当被摄主体偏离焦平面时，光线在传感器前后形成"弥散圆（Circle of Confusion, CoC）"，导致影像模糊。

**弥散圆直径**是判断"清晰"与"模糊"的阈值。人眼在标准观看距离下（约25cm，相当于8x10英寸打印），弥散圆直径不超过 0.03mm（全画幅）时即视为清晰。这也是景深计算的基础参数。

### 1.2 自动对焦模式

| AF模式 | 工作方式 | 适用场景 |
|--------|----------|----------|
| **单次自动对焦（AF-S/ONE SHOT）** | 半按快门锁定焦点，锁定后不追焦 | 静态主体、人像摆拍、风光、静物 |
| **连续自动对焦（AF-C/AI SERVO）** | 持续追踪移动主体，快门释放时最终确认焦点 | 运动、野生动物、奔跑的儿童 |
| **自动切换（AF-A/AI FOCUS）** | 主体静止用单次，主体移动自动切连续 | 不确定主体是否运动的场合 |
| **手动对焦（MF）** | 摄影师通过旋转对焦环精确控制 | 微距、星空、低对比度场景、创意对焦 |

### 1.3 对焦点选择策略

- **单点对焦**：精确控制焦点位置，适合人像（对眼睛）、微距
- **动态区域对焦**：选定的主对焦点与周围辅助点协同追踪，适合不规则运动的主体
- **自动区域对焦**：相机自动判断最近/反差最大的主体，适合快速抓拍
- **面部/眼部检测对焦**：现代相机标配，优先识别并锁定人眼，成功率极高

### 1.4 对焦精度问题

- **前移/后移（Front/Back Focus）**：镜头实际焦点偏离预期位置，需要通过机身微调（AF Fine Tune）校正
- **跑焦**：连续对焦时焦点漂移到背景或其他物体上，通常是因为对焦区域过大或对比度不足
- **呼吸效应（Focus Breathing）**：改变对焦距离时视角随之变化，视频拍摄中尤为明显

## 二、景深理论

### 2.1 景深的定义

景深（Depth of Field, DOF）是指在焦平面前后，影像被判定为"可接受清晰"的距离范围。

```
                    景深
|←————— 后景深 —————|焦平面|————— 前景深 ———→|
|                    |      |                    |
近处模糊  ·····  清晰开始 ────── 焦点 ────── 清晰结束  ·····  远处模糊
```

**关键特性：前景深通常小于后景深，比例约为 1:2。** 即焦点对在某个位置时，其前方清晰范围约为后方清晰范围的一半。

### 2.2 影响景深的四大因素

| 因素 | 变化方向 | 景深变化 | 原理 |
|------|----------|----------|------|
| **光圈** | 光圈越大（f值越小） | 景深越浅 | 弥散圆随孔径增大而增大 |
| **焦距** | 焦距越长 | 景深越浅 | 长焦放大弥散圆 |
| **对焦距离** | 距离越近 | 景深越浅 | 近距离放大率使弥散圆更显著 |
| **传感器画幅** | 画幅越大 | 景深越浅（相同视角下） | 大画幅需要更长焦距或更大光圈获得相同视角 |

#### 量化关系

在相同构图下（即被摄主体在画面中占相同比例）：

- **光圈每开大一档**：景深约减少 30-40%
- **焦距翻倍**：景深约减少为原来的 1/4
- **对焦距离减半**：景深约减少为原来的 1/4

### 2.3 景深控制实战参数表

以全画幅相机为例，对焦距离约 3 米：

| 焦距 | f/1.4 | f/2.8 | f/5.6 | f/8 | f/11 |
|------|-------|-------|-------|-----|------|
| 24mm | ∞-0.9m | ∞-1.8m | ∞ | ∞ | ∞ |
| 35mm | 4.5m | 2.3m | 9.1m | ∞ | ∞ |
| 50mm | 0.36m | 0.70m | 1.4m | 2.1m | 3.2m |
| 85mm | 0.12m | 0.24m | 0.47m | 0.71m | 1.04m |
| 135mm | 0.06m | 0.12m | 0.24m | 0.36m | 0.52m |
| 200mm | 0.03m | 0.05m | 0.11m | 0.16m | 0.24m |

> 表中数值为**总景深**（前景深 + 后景深），单位为米。

### 2.4 景深预览功能

多数单反和无反相机提供"景深预览按钮"，按下时光圈收缩到设定值，可在取景器中直接观察实际景深效果。缺点是光圈较小时取景器会变暗。

更推荐的方式：
- 使用相机的**景深模拟显示**（多数无反相机支持）
- 拍摄后回放检查
- 使用手机APP远程取景查看

## 三、超焦距

### 3.1 超焦距定义

当镜头对焦在超焦距（Hyperfocal Distance, H）处时，景深从 H/2 延伸到无穷远。这是风光摄影中获得"前景到无穷远都清晰"的核心技术。

### 3.2 超焦距计算公式

```
H = f² / (N × c)

其中：
H = 超焦距（mm）
f = 镜头焦距（mm）
N = 光圈f值（如 f/8 时 N=8）
c = 弥散圆直径（全画幅 0.03mm，APS-C 0.02mm）
```

### 3.3 常用超焦距速查表（全画幅，c=0.03mm）

| 焦距 | f/5.6 | f/8 | f/11 | f/16 |
|------|-------|-----|------|------|
| 16mm | 1.53m | 1.07m | 0.78m | 0.54m |
| 24mm | 3.43m | 2.40m | 1.75m | 1.20m |
| 35mm | 7.29m | 5.10m | 3.72m | 2.56m |
| 50mm | 14.88m | 10.42m | 7.58m | 5.21m |

**使用方法**：对焦在表中的超焦距数值处，则从该距离的一半到无穷远都是清晰的。

例如：24mm 镜头 + f/11 → 超焦距 1.75m → 对焦在 1.75m 处 → 从 0.88m 到无穷远全部清晰。

### 3.4 超焦距的注意事项

1. **弥散圆标准影响**：网络上的超焦距计算器有的使用较严格的弥散圆标准（0.03mm），有的较宽松（0.025mm 或 0.02mm），结果会有差异
2. **像素密度考量**：高像素机身（4500万+）放大查看时对弥散圆更敏感，建议适当缩小弥散圆参数或收缩半档光圈
3. **镜头素质**：实际锐度在最佳光圈附近最优，盲目追求超焦距而使用 f/22 可能因衍射降低整体画质

## 四、前景虚化（散景）控制

### 4.1 散景（Bokeh）的美学

散景不仅仅是"虚化"，更指虚化区域的**质量**。好的散景应该：
- 高光虚化斑呈**圆形**（而非多边形）
- 过渡**柔和均匀**，没有明显的"洋葱圈"或"甜甜圈"效果
- 前景和背景虚化过渡**自然渐进**

### 4.2 影响虚化程度的因素

虚化强度取决于背景虚化的**模糊量**，近似公式：

```
模糊量 ≈ (|对焦距离 - 背景距离| / 对焦距离) × (焦距² / (光圈值 × 对焦距离))
```

**获得强烈虚化的操作策略**：

| 策略 | 操作 | 效果 |
|------|------|------|
| **最大光圈** | 使用 f/1.4 - f/2.0 镜头 | 虚化程度直接倍增 |
| **长焦距** | 使用 85mm、135mm、200mm | 压缩空间、放大虚化 |
| **缩短对焦距离** | 靠近被摄主体 | 前景虚化显著增强 |
| **拉大背景距离** | 让主体远离背景 | 背景虚化程度增加 |

### 4.3 光圈叶片数量与散景形状

- **7片圆形光圈叶片**：虚化光斑近圆形，散景柔和（如索尼 FE 85mm f/1.4 GM）
- **9片圆形光圈叶片**：光斑更加圆润，在大光圈下尤其明显（如佳能 RF 85mm f/1.2 L）
- **直边光圈叶片**：收小光圈时虚化光斑呈多边形

### 4.4 前景虚化的创意运用

1. **前景遮挡物**：利用花朵、树叶、栏杆等作为前景，开大光圈使其虚化为色块，增加画面层次
2. **前景框架**：虚化的前景形成自然框架，引导视线到主体
3. **前景色块**：利用前景虚化创造色彩氛围（如黄色虚化树叶营造秋日感）

## 五、不同题材的景深策略

### 5.1 人像摄影

- **特写/半身**：f/1.4 - f/2.8，确保双眼都在景深范围内（眼睛到耳朵的距离约 5cm，在 85mm f/1.4 对焦 2m 处景深仅约 3cm，需精确对焦在近侧眼）
- **全身/环境人像**：f/4 - f/8，保证全身清晰同时有适当背景分离
- **多人合影**：f/5.6 - f/11，排列成弧线而非直线，确保所有人都清晰

### 5.2 风光摄影

- **大景深风光**：f/8 - f/11（多数镜头最佳画质区间），配合超焦距技术
- **极远景（山脉/远景无前景）**：f/5.6 - f/8 即可（对焦在远处时景深已很大）
- **近景风光（有前景）**：f/11 - f/16，可能需要景深合成（Focus Stacking）

### 5.3 微距摄影

- 微距下景深极浅：1:1 放大比下，f/2.8 的景深可能不到 1mm
- 常用 f/8 - f/16 来增加景深，但仍可能不够
- **景深合成**是微距解决景深不足的标准方案：拍摄多张不同对焦距离的照片后合成

### 5.4 街头摄影

- **经典方案**：f/5.6 - f/8，预设对焦距离（区域对焦），利用景深覆盖一定范围
- **选择性对焦**：f/2 - f/2.8，精确对焦单个主体，虚化人群创造视觉分离

### 5.5 星空摄影

- 几乎无限远的对焦距离 → 原本景深应很大
- 但大光圈 + 广角 + 星点弥散 → 需要对焦精确
- **对焦方法**：开启实时取景放大，对准一颗亮星手动对焦至最小光点

## 六、景深合成（Focus Stacking）

### 6.1 适用场景

当单张拍摄无法满足景深要求时（微距、极近距离风光），通过拍摄多张不同焦点的照片，后期合成获得全程清晰图像。

### 6.2 拍摄流程

1. 固定相机（必须使用三脚架）
2. 设定合适的固定光圈（通常 f/5.6 - f/8，不要太小）
3. 从最近对焦点开始，每张微调对焦距离（可使用对焦轨道或手动旋转对焦环）
4. 确保相邻两张的景深有 20-30% 的重叠
5. 保持相同曝光参数，使用快门线或延迟快门

### 6.3 后期合成工具

- **Helicon Focus**：专业景深合成软件，算法成熟
- **Zerene Stacker**：微距领域首选
- **Photoshop**：自动混合图层功能，操作简单但复杂场景效果不如专业软件
- **Affinity Photo**：自带景深合成功能

## 七、对焦与景深的常见错误

| 错误 | 表现 | 解决方案 |
|------|------|----------|
| **对焦在鼻尖而非眼睛** | 人像中鼻尖清晰而眼睛模糊 | 选择单点对焦，精确锁定眼睛 |
| **光圈过大导致一只眼清晰** | 侧面人像中近侧眼清晰、远侧眼模糊 | 适当收小光圈至 f/2.8，或调整角度使双眼等距 |
| **风光忘记检查前景** | 远景锐利但前景模糊 | 使用超焦距或景深合成 |
| **微距相机移动** | 拍摄过程中因呼吸/手持导致焦点偏移 | 使用三脚架 + 快门线，或开启焦点包围拍摄 |
| **大光圈衍射** | f/22 拍摄发现整体画质下降 | 使用 f/8-f/11 + 景深合成代替极小光圈 |
