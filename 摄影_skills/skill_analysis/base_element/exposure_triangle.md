# 曝光三要素：光圈、快门、ISO

## 一、核心概念

曝光三角（Exposure Triangle）是摄影最根本的理论框架，由三个相互关联的参数构成：

| 参数 | 物理含义 | 控制效果 | 单位/表示 |
|------|----------|----------|-----------|
| **光圈（Aperture）** | 镜头通光孔径大小 | 进光量 + 景深 | f/数值（如 f/2.8、f/8） |
| **快门（Shutter）** | 传感器感光时间长度 | 进光量 + 运动模糊/凝固 | 时间（如 1/250s、2s） |
| **ISO（感光度）** | 传感器对光的敏感程度 | 画面亮度 + 噪点水平 | 数值（如 ISO 100、3200） |

### 关键原则
**曝光值（EV）= 光圈贡献 + 快门贡献 + ISO贡献**

三者中任意一个变化，都需要至少另一个的补偿来维持相同的曝光量。这不是简单的"加减法"，而是遵循**互易律（Reciprocity Law）**。

## 二、互易关系详解

### 2.1 光圈档位体系

光圈值（f-number）是焦距与有效孔径的比值：

```
f/1.0  → f/1.4  → f/2.0  → f/2.8  → f/4.0  → f/5.6  → f/8.0  → f/11  → f/16  → f/22  → f/32
```

**每相邻一档，进光量相差一倍。** f/2.8 的进光量是 f/4.0 的 2 倍，是 f/5.6 的 4 倍。

- **大光圈**（f/1.4 - f/2.8）：进光多、景深浅、背景虚化强
- **中光圈**（f/4 - f/8）：多数镜头最佳画质区间、景深适中
- **小光圈**（f/11 - f/22）：进光少、景深大、可能产生衍射

### 2.2 快门速度档位体系

```
1/8000 → 1/4000 → 1/2000 → 1/1000 → 1/500 → 1/250 → 1/125 → 1/60 → 1/30 → 1/15 → 1/8 → 1/4 → 1/2 → 1s → 2s → 4s → ...
```

**每相邻一档，进光量相差一倍。**

- **高速快门**（≥1/500s）：凝固运动瞬间（体育、鸟类飞行）
- **中速快门**（1/60 - 1/250s）：日常手持拍摄的安全区间
- **低速快门**（≤1/30s）：需要三脚架，用于创意模糊（流水、车轨、星轨）

**安全快门法则**：手持拍摄时，快门速度的分母应 ≥ 焦距。例如 50mm 镜头建议不低于 1/50s，200mm 镜头建议不低于 1/200s。带防抖（IS/VR/OS）时可放宽 2-5 档。

### 2.3 ISO 档位体系

```
ISO 50 → ISO 100 → ISO 200 → ISO 400 → ISO 800 → ISO 1600 → ISO 3200 → ISO 6400 → ISO 12800 → ISO 25600
```

**每翻倍一档，进光量等效增加一倍（或信号放大一倍）。**

- **低 ISO**（50-200）：画质最纯净、动态范围最大
- **中 ISO**（400-1600）：可接受轻微噪点
- **高 ISO**（≥3200）：噪点明显增加，细节损失

### 2.4 等效曝光组合示例

以下组合产生完全相同的曝光亮度：

| 组合 | 光圈 | 快门 | ISO | 适用场景 |
|------|------|------|-----|----------|
| A | f/2.8 | 1/500s | 100 | 运动人像，浅景深 |
| B | f/5.6 | 1/125s | 100 | 一般场景，中景深 |
| C | f/8.0 | 1/30s | 100 | 风光（需三脚架），大景深 |
| D | f/2.8 | 1/1000s | 200 | 室内运动，低光 |
| E | f/5.6 | 1/60s | 400 | 室内活动 |

### 2.5 互易律失效

在极长曝光（通常超过 30 秒至数分钟）时，胶片和部分数字传感器的互易律会"失效"——实际需要的曝光量比理论计算值更多。数码相机中长曝降噪功能可缓解此问题，但需要等待与曝光时间等长的降噪处理。

## 三、直方图解读

### 3.1 直方图基本结构

直方图是判断曝光是否正确的核心工具，横轴表示亮度（0=纯黑，255=纯白），纵轴表示该亮度区间的像素数量。

```
|  ｜
|  ｜    ｜
|  ｜ ｜ ｜
|  ｜ ｜ ｜ ｜
|__｜_｜__｜__｜__ 亮度
  暗    中    高
```

### 3.2 直方图形态分析

| 形态 | 描述 | 曝光状态 | 处理建议 |
|------|------|----------|----------|
| **正常分布** | 山峰集中在中间区域，两侧平滑下降 | 曝光正确 | 维持当前参数 |
| **左偏（暗调主导）** | 山峰集中在左侧 | 可能欠曝 | 提升曝光补偿，观察暗部细节 |
| **右偏（亮调主导）** | 山峰集中在右侧 | 可能过曝 | 降低曝光补偿，检查高光 |
| **双峰** | 两个明显峰值 | 高反差场景（逆光、剪影） | 根据创作意图决定 |
| **右侧撞墙** | 右端像素堆积且被截断 | **高光溢出（过曝死白）** | 必须减曝光，死白区域无法恢复 |
| **左侧撞墙** | 左端像素堆积且被截断 | **暗部溢出（欠曝死黑）** | 适当加曝光，纯黑区域同样无法恢复 |

### 3.3 "向右曝光"（ETTR）策略

由迈克尔·赖克曼（Michael Reichmann）提出的曝光策略：

- 在不使高光溢出的前提下，尽可能将曝光推向右侧
- 数码传感器在亮部区域记录的数据比暗部区域多得多
- 后期提亮暗部会放大噪点，而压暗亮部噪点几乎不增加
- **实践方法**：开启"高光警告（闪烁过曝提示）"，逐渐增加曝光直到刚刚出现极少量的高光溢出，然后回调 1/3 档

### 3.4 各场景直方图参考

- **人像**：山峰偏右中，皮肤色调不应溢出
- **风光**：应充分利用全亮度范围，注意天空不要溢出
- **夜景/星空**：整体偏左是正常的，但主体不应完全死黑
- **雪景/白墙**：画面以亮部为主，直方图偏右，需 +0.7 至 +1.3 EV 曝光补偿（"白加黑减"原则）

## 四、常见曝光问题与解决方案

### 4.1 过曝（Overexposure）

**表现**：画面整体偏亮、高光区域纯白无细节、色彩发白

**原因**：
- 光圈过大或快门过慢
- ISO 设置过高
- 测光模式选择不当（如在雪景中用评价测光）
- 曝光补偿未调低

**解决方案**：
1. 减少曝光补偿（-0.3 至 -2.0 EV）
2. 缩小光圈或提高快门速度
3. 检查测光模式，必要时切换到点测光对中灰区域测光
4. 开启高光警告实时监控

### 4.2 欠曝（Underexposure）

**表现**：画面整体偏暗、暗部纯黑无细节、噪点严重（后期提亮时）

**原因**：
- 光圈过小或快门过快
- 逆光拍摄未加曝光补偿
- ISO 过低

**解决方案**：
1. 增加曝光补偿（+0.3 至 +2.0 EV）
2. 开大光圈或降低快门速度
3. 适当提高 ISO
4. 使用反光板或闪光灯补光

### 4.3 混合光场景曝光困难

**表现**：场景中明暗反差极大（如逆光人像、室内窗户边），无论怎么调都有一部分无法兼顾

**解决方案**：
1. **HDR（高动态范围）**：包围曝光多张合成
2. **渐变灰滤镜（GND）**：压暗亮部（风光摄影）
3. **闪光灯补光**：提亮暗部（人像摄影）
4. **RAW格式拍摄**：后期拉回最多 2-3 档的过曝/欠曝细节

### 4.4 震动模糊（相机抖动）

**表现**：整个画面出现方向性模糊，尤其是暗光、长焦距场景

**解决方案**：
1. 提高快门速度至安全快门以上
2. 使用三脚架或独脚架
3. 开启镜头/机身防抖
4. 采用正确的持机姿势（肘部抵住身体、半呼吸时按快门）

### 4.5 运动模糊

**表现**：主体模糊而背景清晰（快门速度不够凝固运动）

**解决方案**：
1. 提高快门速度（人物步行至少 1/125s，跑步 1/500s，体育 1/1000s+）
2. 开大光圈
3. 提高 ISO
4. 预对焦到运动轨迹上的某个点，等待主体进入

## 五、曝光模式选择指南

| 模式 | 摄影师控制 | 相机控制 | 适用场景 |
|------|------------|----------|----------|
| **A/Av（光圈优先）** | 光圈、ISO | 快门速度 | 人像、风光、静物——需要控制景深 |
| **S/Tv（快门优先）** | 快门、ISO | 光圈 | 运动、体育、鸟类——需要控制运动凝固/模糊 |
| **M（全手动）** | 光圈、快门、ISO | 无 | 影棚、长曝、星空——光线条件稳定或需要精确控制 |
| **P（程序自动）** | 少量参数调节 | 光圈和快门 | 日常抓拍、新手入门 |
| **Auto（全自动）** | 仅构图 | 全部 | 完全新手或紧急记录 |

### 推荐工作流

1. **人像**：A/Av 模式 → 选择最大光圈 → ISO 自动 → 曝光补偿 ±0.3
2. **风光**：A/Av 模式 → f/8-f/11 → ISO 100 → 三脚架 + 2秒延迟快门
3. **运动**：S/Tv 模式 → 1/500s 以上 → ISO 自动 → 连续对焦 + 高速连拍
4. **星空**：M 模式 → f/2.8 → 20-25秒 → ISO 3200-6400 → 手动对焦至无限远回调
5. **街拍**：A/Av 模式 → f/5.6-f/8（区域对焦）→ ISO 自动 → 区域对焦预设距离

## 六、进阶概念

### 6.1 曝光补偿与测光模式

- **评价测光（矩阵测光）**：综合全画面亮度，适合均匀光照场景
- **中央重点测光**：偏重画面中央区域，兼顾四周，适合经典人像
- **点测光**：仅测量取景器中央 1-5% 区域，适合逆光、舞台、高反差场景
- **局部测光**：测量中央约 6-10% 区域，介于点测光与中央重点之间

### 6.2 动态范围（Dynamic Range）

人眼动态范围约 20-24 档，而数码相机通常为 12-15 档。这意味着相机无法同时记录人眼所见的全部亮度范围。理解这一限制是掌握曝光的核心前提。

### 6.3 档位与 EV 值对照

EV 0 = f/1.0 + 1s + ISO 100。每增加一 EV，需要减少一档光圈、或提高一档快门、或降低一档 ISO。

晴天户外（f/16 + 1/125s + ISO 100）约为 EV 15，月光场景约为 EV -3 至 -6。

### 6.4 "阳光十六法则"

晴天户外，设 ISO 100、f/16、快门速度为 1/100s（约等于 ISO 值的倒数），可获得正确曝光。这是无需测光表的经验法则，衍生规则：

| 光线条件 | 光圈 | 快门（ISO 100） |
|----------|------|-----------------|
| 晴天（有明显阴影） | f/16 | 1/100s |
| 多云（软阴影） | f/11 | 1/100s |
| 阴天（无阴影） | f/8 | 1/100s |
| 日出/日落 | f/5.6 | 1/100s |
