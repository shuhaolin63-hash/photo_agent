# 摄影诊断AI Agent系统 - 场景库说明

> 版本：1.0.0 | 最后更新：2026-07-13
> 所属模块：任务调度 - 场景识别库（scene_lib）

---

## 概述

场景库是摄影诊断系统的核心分类基础，定义了系统支持的**五大摄影场景**及其特征描述、判断关键词和子场景分类。场景识别由CLIP模型自动完成，本库提供辅助判断规则和关键词表。

---

## 一、人像摄影（portrait）

### 场景定义

以人物为主要表现主体的摄影形式。人物在画面中占据显著位置，面部或肢体是视觉焦点，旨在展现人物的外貌特征、性格特质、情绪状态或与环境的叙事关系。

### 核心特征

| 特征维度 | 描述 |
|---------|------|
| **主体识别** | 画面中存在清晰可见的人脸/人物，面部面积通常 >= 画面5% |
| **构图特点** | 人物位于视觉重心区域，背景为辅助或衬托作用 |
| **光线特征** | 注重面部光线质量，常用伦勃朗光、蝴蝶光、环形光等 |
| **景深控制** | 多使用浅景深（大光圈）突出主体，虚化背景 |
| **情绪表达** | 通过表情、眼神、姿态传达情感或故事 |

### 判断关键词

**CLIP正向标签：**
`portrait photo` / `person close-up` / `face portrait` / `studio portrait` / `outdoor portrait` / `couple photo` / `group portrait` / `wedding photo` / `fashion photography` / `environmental portrait`

**CLIP负向标签：**
`landscape without people` / `empty room` / `food on table` / `night sky` / `product photo` / `macro photography`

**文本匹配关键词：**
- P1强匹配：人像、肖像、写真、证件照、婚纱照、情侣照、全家福、脸、面部、表情、眼神
- P2中匹配：半身照、全身照、特写、大头照、合照、闺蜜照、毕业照、cosplay、模特
- P3弱匹配：人、拍摄对象、人物照

### 子场景分类

| 子场景ID | 名称 | 判断依据 |
|---------|------|---------|
| `portrait.studio` | 影棚人像 | 纯色/渐变背景、可控光源、无环境信息 |
| `portrait.environmental` | 环境肖像 | 人物与环境有叙事关联，环境信息丰富 |
| `portrait.outdoor` | 户外人像 | 自然光/混合光，户外场景（公园/街道/建筑等） |
| `portrait.candid` | 抓拍人像 | 人物处于自然状态， unaware of camera |
| `portrait.couple` | 情侣/双人照 | 两人互动构图，情感关联明显 |
| `portrait.group` | 合影/团体照 | 3人以上，排列整齐或自然组合 |
| `portrait.ceremony` | 仪式人像 | 婚礼/毕业/庆典等正式场合人像 |

### 典型构图模式

- 三分法面部放置（双眼位于上三分之一线）
- 中心对称构图（证件照/正式肖像）
- 黄金螺旋构图（环境肖像/艺术人像）
- 留白构图（侧面肖像+大面积留白传达情绪）
- 框架构图（利用门框/窗户/树枝等自然框）

## 二、风光摄影（landscape）

### 场景定义

以自然景观或城市景观为表现主体的摄影形式。画面以广阔的空间、丰富的层次和宏大的气势为主要特征，注重光影氛围、色彩层次和空间感的营造。

### 核心特征

| 特征维度 | 描述 |
|---------|------|
| **主体识别** | 画面以自然/城市景观为主体，天际线/地平线明显 |
| **构图特点** | 前景/中景/远景层次分明，注重引导线和视觉纵深 |
| **光线特征** | 强调时段光线（黄金时段/蓝调时刻/晨雾/暮光） |
| **色彩特征** | 天空与地面的色彩过渡，大气透视效果 |
| **景深控制** | 多使用小光圈获得大景深，保持前后清晰 |

### 判断关键词

**CLIP正向标签：**
`landscape photography` / `mountain scenery` / `ocean sunset` / `city skyline` / `aerial photography` / `waterfall photo` / `forest scene` / `desert landscape` / `rainbow over field` / `panoramic view`

**CLIP负向标签：**
`close-up face` / `studio product shot` / `indoor still life` / `food plate` / `macro insect` / `portrait background`

**文本匹配关键词：**
- P1强匹配：风景、风光、山川、大海、天空、云海、日出、日落、城市风光、建筑、航拍
- P2中匹配：全景、黄昏、清晨、海边、湖面、田野、草原、雪山、瀑布、森林、沙漠
- P3弱匹配：景色、美景、旅游照、外景、自然

### 子场景分类

| 子场景ID | 名称 | 判断依据 |
|---------|------|---------|
| `landscape.mountain` | 山岳风光 | 山体占据画面主体，可见山峰/山脊线 |
| `landscape.ocean` | 海景风光 | 海面/海岸线为主体，可见浪花/礁石 |
| `landscape.sunrise_sunset` | 日出日落 | 太阳可见或位于地平线附近，极端暖色温 |
| `landscape.city` | 城市风光 | 建筑/天际线为主体，城市结构清晰 |
| `landscape.forest` | 森林/树木 | 树木/林间光影为主体，绿色为主色调 |
| `landscape.waterfall` | 瀑布/流水 | 流水为主体，可见动态模糊效果 |
| `landscape.desert` | 沙漠/荒原 | 沙丘/戈壁为主体，暖色调/极简构图 |
| `landscape.aerial` | 航拍风光 | 俯视角度>60度，鸟瞰视角 |

### 典型构图模式

- 前景-中景-远景三层构图（经典风光构图法）
- 引导线构图（道路/河流/栅栏引导视觉深入画面）
- 对称构图（水面倒影形成上下对称）
- 天际线三分法（天空占1/3或2/3）
- 极简留白构图（大面积天空/雪地/水面）

## 三、纪实摄影（documentary）

### 场景定义

以记录真实事件、人物生活状态和社会现象为目的的摄影形式。强调"决定性瞬间"的捕捉，注重叙事性、现场感和真实性的表达，反对摆拍和人为干预。

### 核心特征

| 特征维度 | 描述 |
|---------|------|
| **主体识别** | 画面捕捉"正在发生的事件"或"人物真实生活状态" |
| **构图特点** | 强调时机和时机感，构图可能不规则/打破规则 |
| **光线特征** | 多为现场光，不刻意控制光线 |
| **叙事性** | 画面本身传达故事或情感，具有信息密度 |
| **真实感** | 人物表情自然，场景真实，无摆拍痕迹 |

### 判断关键词

**CLIP正向标签：**
`street photography` / `documentary photo` / `candid photo` / `news photo` / `travel photography` / `market scene` / `festival crowd` / `sports action` / `daily life scene` / `protest rally`

**CLIP负向标签：**
`studio portrait` / `posed photo` / `arranged still life` / `food styling` / `product advertising` / `fashion model standing still`

**文本匹配关键词：**
- P1强匹配：纪实、街拍、新闻、人文、旅行、节庆、民俗、运动、纪实摄影
- P2中匹配：街头、市场、集市、人群、游行、比赛、演唱会、婚礼纪实、现场
- P3弱匹配：生活、日常、故事、瞬间、记录

### 子场景分类

| 子场景ID | 名称 | 判断依据 |
|---------|------|---------|
| `documentary.street` | 街头摄影 | 户外公共场所，人物自然状态，非正式 |
| `documentary.news` | 新闻摄影 | 事件现场，新闻价值高，时效性强 |
| `documentary.travel` | 旅行纪实 | 异域文化场景，标志性地点，旅途记录 |
| `documentary.sports` | 运动纪实 | 运动场景，动态瞬间，竞技场面 |
| `documentary.social` | 社会纪实 | 社会现象记录，人文关怀，群体生活 |
| `documentary.culture` | 文化民俗 | 传统仪式/民俗活动/非物质文化遗产 |
| `documentary.food_culture` | 美食文化 | 食物制作过程/市集/饮食文化（非摆拍美食） |

### 典型构图模式

- 决定性瞬间构图（布列松理论，动作与构图的完美结合）
- 环境融入构图（人物与环境共同构成叙事）
- 层次构图（前景人物+中景活动+背景环境）
- 对角线构图（增加动态感和张力）
- 框中框构图（门洞/窗户/人群间隙形成自然框）

## 四、星空摄影（starry）

### 场景定义

以夜空中的天体（星星/银河/星轨/极光/月亮等）为主要表现对象的摄影形式。通常在极暗光环境下使用长时间曝光或高ISO拍摄，技术门槛较高，涉及追踪赤道仪、堆栈合成等专业技术。

### 核心特征

| 特征维度 | 描述 |
|---------|------|
| **主体识别** | 画面主体为天空中的天体，星点/银河/星轨/极光清晰可见 |
| **构图特点** | 天空为主体，地景为前景衬托或剪影 |
| **光线特征** | 极暗光环境，自然光仅有月光/星光/极光 |
| **技术特征** | 长曝光/高ISO/大光圈/堆栈合成/赤道仪追踪 |
| **色彩特征** | 深蓝/深紫色天空，银河核心偏暖，极光为绿/紫色 |

### 判断关键词

**CLIP正向标签：**
`milky way photo` / `star trail photography` / `northern lights aurora` / `astrophotography` / `night sky stars` / `meteor shower` / `lunar eclipse` / `solar eclipse` / `constellation photo` / `deep sky object`

**CLIP负向标签：**
`daytime landscape` / `city lights without stars` / `portrait outdoor` / `indoor scene` / `product photography`

**文本匹配关键词：**
- P1强匹配：星空、银河、星轨、极光、流星、天文摄影、星空摄影
- P2中匹配：夜空、星星、月亮、月蚀、日蚀、星云、深空、星座
- P3弱匹配：夜景、夜晚、暗光、长时间曝光、光绘

### 子场景分类

| 子场景ID | 名称 | 判断依据 |
|---------|------|---------|
| `starry.milky_way` | 银河摄影 | 可见银河弧形结构，星云核心区域可见 |
| `starry.star_trail` | 星轨摄影 | 星点呈现弧线轨迹，长曝光合成 |
| `starry.aurora` | 极光摄影 | 天空可见绿色/紫色光带，动态形态 |
| `starry.deep_sky` | 深空天文 | 星云/星团/星系特写，无地景，望远镜拍摄 |
| `starry.night_landscape` | 夜景星空 | 星空+地景结合，前景为地面景观剪影 |

### 典型构图模式

- 银河拱桥构图（银河弧形横跨画面）
- 放射构图（以北极星为中心的同心圆星轨）
- 前景星空构图（地景前景+星空背景，增加空间感）
- 极简星空构图（大面积星空+极简地平线）
- 人物星空构图（小人物+大星空，强调宇宙与人的对比）

### 技术判断辅助规则

| 指标 | 星空典型值 | 用途 |
|------|-----------|------|
| 快门速度 | 10s-25s（500法则） | 区分星空与普通夜景 |
| ISO | 1600-6400 | 判断是否为星空拍摄参数 |
| 光圈 | f/1.4-f/2.8 | 大光圈星空特征 |
| 色温 | 3500K-4500K | 夜空典型色温 |
| 画面暗区占比 | > 60% | 星空照片通常大面积深色 |

## 五、静物摄影（still_life）

### 场景定义

以无生命物体为主要表现对象的摄影形式。包括美食、产品、花卉、珠宝等，通常在受控环境下精心布置，注重质感、色彩和光影的精确呈现，广泛应用于商业广告、美食杂志、产品目录等领域。

### 核心特征

| 特征维度 | 描述 |
|---------|------|
| **主体识别** | 画面主体为无生命物体，无脸部/人物主体 |
| **构图特点** | 精心布置的主体位置，注重平衡和美感 |
| **光线特征** | 可控光源（柔光箱/反光板/聚光灯），强调质感呈现 |
| **背景处理** | 背景干净或有特定设计，不干扰主体 |
| **质感表现** | 强调物体表面的纹理、光泽、材质特征 |

### 判断关键词

**CLIP正向标签：**
`still life photography` / `food photography` / `product photography` / `macro flower` / `jewelry close-up` / `watch product shot` / `table setting` / `flat lay photo` / `arranged objects` / `commercial product shot`

**CLIP负向标签：**
`outdoor landscape` / `street scene` / `portrait face` / `night sky` / `candid photo` / `action sports`

**文本匹配关键词：**
- P1强匹配：静物、微距、美食摄影、产品摄影、花卉摄影、珠宝摄影、手表摄影
- P2中匹配：食物摆盘、桌面、茶几、书桌、橱窗、商品照、广告片、菜谱配图、茶艺
- P3弱匹配：特写、细节、质感、纹理、摆拍、道具、工艺品

### 子场景分类

| 子场景ID | 名称 | 判断依据 |
|---------|------|---------|
| `still_life.food` | 美食摄影 | 食物/餐具/烹饪场景为主，注重食欲感 |
| `still_life.product` | 产品摄影 | 商品为主体，背景干净，商业展示用途 |
| `still_life.flower` | 花卉静物 | 花朵为主体，有布置/裁剪/组合痕迹 |
| `still_life.jewelry` | 珠珠宝石 | 珠宝/首饰/宝石特写，强调光泽和切工 |
| `still_life.flat_lay` | 平铺构图 | 俯拍视角，物品平铺排列（桌面/布置） |
| `still_life.craft` | 工艺品 | 手工制品/古董/收藏品，强调质感与文化 |

### 典型构图模式

- 中心对称构图（产品/美食居中，背景均匀）
- 斜线/三角构图（多物品组合，形成稳定三角形）
- 俯拍平铺构图（flat lay，物品有序排列在平面上）
- 留白构图（大面积纯色/渐变背景衬托主体）
- 负空间构图（主体偏一侧，大面积留白增加高级感）

### 布光方案判断辅助

| 布光类型 | 特征描述 | 适用子场景 |
|---------|---------|-----------|
| 柔光箱侧光 | 柔和阴影，质感细腻 | 美食、产品 |
| 聚光灯硬光 | 强对比阴影，戏剧感 | 珠宝、工艺品 |
| 背光/逆光 | 轮廓光效果，透明感 | 饮品、花卉 |
| 漫反射光 | 无明显阴影方向，均匀照明 | 平铺、白底产品 |
| 自然窗光 | 单侧柔和光，自然感 | 美食、花卉 |

## 注意事项

1. 场景分类结果影响后续全部诊断流程，需确保分类准确性
2. 对于跨场景照片（如环境肖像），采用主场景优先策略
3. 场景库需定期更新以适应新的摄影风格和流派
4. 所有场景特征提取均支持本地离线处理（CLIP+OpenCV）
5. 大师作品数据库的更新需遵循版权合规要求
