## 对比维度分析

### 按光线风格分类

| 光线风格 | 代表大师 | 关键特征 |
|----------|----------|----------|
| **精确侧光（伦勃朗光）** | 卡什 | 主光 45° 侧上方，对侧三角光斑，光比 3:1-4:1 |
| **正面柔光（平面化）** | 阿维顿、潘 | 大面积柔光箱正面照射，均匀无方向性阴影 |
| **自然光为主** | 西夫、纽曼、桑德、韦斯顿 | 窗户光/散射光，保留自然质感 |
| **多灯环境光** | 莱博维茨 | 3-8 盏灯，融合环境光与闪光灯 |
| **直接闪光灯** | 阿勃丝 | 闪光灯直射，平面化，无方向阴影 |
| **模仿性光线** | 舍曼 | 根据主题模仿不同媒介的光线风格 |

### 按背景处理分类

| 背景策略 | 代表大师 | 核心理念 |
|----------|----------|----------|
| **丰富环境背景** | 莱博维茨、纽曼 | 环境讲述人物的故事 |
| **极简纯色背景** | 卡什、阿维顿、潘、桑德 | 消除环境干扰，直面人物 |
| **融入自然环境** | 西夫、韦斯顿 | 人物与自然/城市环境有机融合 |
| **日常生活环境** | 阿勃丝 | 保留"真实"生活场景 |
| **模仿性场景** | 舍曼 | 模拟电影/广告/画作的场景 |

### 按画幅偏好分类

| 画幅 | 代表大师 | 画质/质感特征 |
|------|----------|--------------|
| **8x10 大画幅** | 卡什、阿维顿、潘、桑德、韦斯顿 | 极致细节，极致锐度，接触印相 |
| **4x5 大画幅** | 纽曼 | 高画质，兼顾灵活性 |
| **6x6 中画幅** | 阿勃丝 | 方形画幅，经典肖像比例 |
| **35mm** | 西夫、舍曼（早期） | 灵活、即兴、颗粒感 |
| **中画幅数码** | 莱博维茨 | 高画质 + 现代工作流 |

### 按情感基调分类

| 情感基调 | 代表大师 | 适合的拍摄场景 |
|----------|----------|----------------|
| **庄重/权威** | 卡什、桑德 | 企业高管、学术领袖、政府官员肖像 |
| **戏剧/叙事** | 莱博维茨、纽曼 | 环境人像、杂志封面、故事性肖像 |
| **真实/张力** | 阿维顿、阿勃丝 | 纪实肖像、挑战性主题、个性表达 |
| **优雅/精致** | 潘、西夫 | 时尚人像、艺术肖像、高级感 |
| **纯粹/本质** | 韦斯顿 | 艺术人体、静物、极致工艺追求 |
| **反讽/批判** | 舍曼 | 概念摄影、视觉文化批判、艺术创作 |

## 学习路径建议

### 初学者路径
1. **卡什** → 学习经典布光和人物姿态
2. **潘** → 学习构图精致性和画面平衡
3. **纽曼** → 学习环境与人物的融合
4. **阿维顿** → 学习极简背景的力量

### 进阶路径
5. **莱博维茨** → 学习多灯复杂布光和叙事构图
6. **西夫** → 学习广角人像和自然光运用
7. **韦斯顿** → 学习光影雕塑和极致工艺

### 高级路径
8. **桑德** → 学习类型学思维和系统化拍摄
9. **阿勃丝** → 学习直面挑战和社会批判
10. **舍曼** → 学习概念摄影和视觉文化解构
