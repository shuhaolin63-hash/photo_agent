# 奥古斯特·桑德（August Sander）

## 生平简介

奥古斯特·桑德（1876年11月17日-1964年4月20日），德国摄影师，被誉为"类型学肖像摄影之父"和20世纪最重要的纪实摄影师之一。出生于德国科隆附近的 Herdorf，父亲是木匠，家境贫寒。青年时期在矿山工作，一位叔父资助了他的摄影兴趣。

1890年代开始学习摄影，1901年开设自己的摄影工作室。第一次世界大战服役后回到科隆，开始构思宏大的拍摄计划——系统性地记录德国社会各阶层人物。这一项目后来被称为"二十世纪的人"（Menschen des 20. Jahrhunderts），最终包含超过 600 张肖像，分为 7 大类、45 个子类。纳粹时期因其作品不 conform 纳粹意识形态而受到压制。二战中工作室被炸毁，部分底片被毁。战后继续拍摄直至去世。

## 核心风格

**"我不是在拍这个人的脸，我是在拍这个人的类型。"**

- **类型学肖像**——系统地记录社会各阶层、各职业的人物，建立视觉档案
- **极致的客观记录**——拒绝戏剧性的光线和构图，追求"文献式"的精确和公正
- **社会档案的价值**——用摄影为时代留下"面孔的百科全书"
- 人人平等的尊严——无论是农民还是律师，面包师还是画家，每个人都以同样的庄重面对镜头

## 标志性技法

### 构图特点
- **标准化的构图格式**：绝大多数肖像采用相同或相似的构图模板——头部/上半身居中，占画面约 60-70%
- **正面/微偏角度**：人物正面朝向相机或微偏 10-15°，建立与观者的直接对话
- **自然站姿或坐姿**：人物以其自然的姿态出现，不做刻意造型
- **简洁背景**：深色/中性背景，消除一切干扰，突出人物
- **平视角度**：相机与人物眼睛平齐，传达平等和尊重
- **手部处理**：人物常手持与其职业相关的工具/物品，或自然放置于身体两侧

### 光影手法
- **均匀正面光**：光线均匀照射面部，没有强烈的方向性阴影
- **柔和的窗户光**：大量利用北向窗户的柔和自然光
- **无闪光灯**：完全使用自然光或连续灯光
- **低调/中调**：整体偏暗调，人物从暗背景中清晰浮现
- **"无光风格"**：光线存在但不引人注意——目的是清晰记录而非创造美感

### 色调偏好
- **全部黑白**：黑白强化了"文献性"和"历史感"
- **大画幅银盐质感**：使用大画幅相机，保证极致的细节和锐度
- **丰富的中间调**：灰阶过渡细腻，暗部有细节
- **暗调背景**：深色背景与中等亮度的人物形成对比
- **不过度暗房处理**：保持原始曝光的质感，不追求戏剧性的对比度

## 经典参数

| 参数类型 | 推荐设定 |
|----------|----------|
| **相机/画幅** | 13x18cm 或 8x10 大画幅相机 |
| **镜头焦距** | 210-240mm（大画幅标准肖像镜头） |
| **光圈** | f/5.6 - f/11（大画幅保证景深） |
| **快门** | 自然光 + 三脚架，快门速度不限 |
| **胶片** | 大画幅黑白胶片（ISO 100 左右） |
| **后期** | 接触印相（Contact Print）或小尺寸放大 |

## 创作理念

- **"纯粹的观察，不加评判"**——桑德追求的是记录而非解读
- **"每一张面孔都是时代的切片"**——类型学肖像的终极目标是构建一个时代的完整面貌
- **"类型学"方法论**：将社会按阶层、职业、年龄、性别等维度分类，每个类别系统拍摄
- **"农民的土地，农民的脸"**——人物的出身和职业塑造了面孔的纹理
- **拒绝画意摄影**：反对当时流行的柔焦、摆拍等"画意"手法，追求直接摄影的精确
- **尊重每一个个体**：无论是社会精英还是底层劳动者，都以同等的庄重拍摄

## "二十世纪的人"分类体系

桑德将作品分为7大类，45个子类：

| 大类 | 子类示例 | 核心对象 |
|------|----------|----------|
| **农民** | 年轻农民、年迈农民、农民家庭 | 农业劳动者的类型 |
| **手工业者** | 铁匠、木匠、面包师 | 传统手艺从业者 |
| **女性** | 女佣、母亲、职业女性 | 各阶层女性 |
| **各行业者** | 律师、医生、商人 | 专业/商业阶层 |
| **艺术家** | 画家、音乐家、作家 | 文化创作者 |
| **城市居民** | 街头人物、乞丐、市民 | 城市各阶层 |
| **最后的人** | 老人、病人、残疾人 | 社会的边缘群体 |

## 对标评价模板

> 这张照片是否以客观公正的态度记录了拍摄对象的类型特征？构图是否标准化到足以与其他同类肖像进行比较？人物是否展现出与其身份相符的尊严？光线是否清晰但不引人注意？

**评分维度**（1-10分）：
- 客观记录性：是否以不加评判的态度记录了人物的真实面貌
- 类型学准确性：是否清晰地展现了该职业/阶层的视觉特征
- 标准化程度：构图、光线、姿态是否足够标准化
- 人物尊严：是否以同等的庄重对待所有拍摄对象
- 历史文献价值：是否为时代留下了有价值的面孔记录
