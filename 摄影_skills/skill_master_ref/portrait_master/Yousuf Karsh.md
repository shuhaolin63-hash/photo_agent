# 尤素福·卡什（Yousuf Karsh）

## 生平简介

尤素福·卡什（1908年12月23日-2002年7月13日），亚美尼亚裔加拿大人，被公认为20世纪最伟大的肖像摄影师之一。出生于奥斯曼帝国时期的马尔丁（今土耳其），1924年为逃避亚美尼亚种族屠杀随叔叔逃往加拿大。

在魁北克的叔叔开设照相馆，卡什从学徒开始学习摄影。1931年前往波士顿跟随肖像摄影大师约翰·H·加罗深造。1932年在渥太华开设自己的工作室。1941年为温斯顿·丘吉尔拍摄的肖像使其一举成名，成为各国政要、文化领袖的首选肖像摄影师。一生拍摄了超过15,000张肖像，其作品被全球150多家博物馆收藏。

## 核心风格

**"我寻找的是人物内心的光辉。"**

- 经典的**工作室肖像**风格——精确控制每一个细节
- 以"**捕捉人物灵魂**"为毕生追求
- 布光严谨、构图庄重、细节精致
- 擅长让拍摄对象在短暂的拍摄中展现其内在性格和成就的重量

## 标志性技法

### 构图特点
- **经典肖像比例**：通常为半身或头部+肩部，人物居中或微偏
- **简洁背景**：深灰/深色背景，消除一切干扰
- **相机角度**：略低角度拍摄，让人物显得庄严、有权威感
- **手臂姿态**：引导拍摄对象的手部做出标志性手势，成为肖像的"签名"
- **直接凝视**：人物通常直视镜头，建立与观者的强烈视觉联系
- **大画幅**：使用8x10英寸大画幅相机，保证极致的细节和画质

### 光影手法
- **经典的"伦勃朗光"布光**：主光在主体前方45°侧上方，在对侧脸颊形成三角形光斑
- **精确的光比控制**：主光与辅光比通常为 3:1 至 4:1，保持立体感但不过度戏剧化
- **发丝光/轮廓光**：从背后/侧后方打一盏灯勾勒头发轮廓，分离主体与背景
- **背景光**：轻微照亮背景，创造层次但不分散注意力
- **最少灯原则**：通常只用 2-3 盏灯，效果精确自然

**卡什标志性布光方案**：
```
          [主灯 - 45° 侧上方]
                ↘
     [辅灯] →  人物  ← [轮廓灯/背景灯]
```

### 色调偏好
- **黑白为主**：一生中绝大部分作品为黑白，强调光影和质感而非色彩
- **丰富的中间调**：暗部有细节，亮部不溢出，灰阶过渡丰富
- **暗调氛围**：背景和阴影偏暗，主体从暗调中"浮现"
- **高对比度**：比一般肖像更强的高光/阴影对比，增加力量感
- **极少彩色作品**：晚年少量彩色，但仍保持沉稳调性

## 经典参数

| 参数类型 | 推荐设定 |
|----------|----------|
| **相机/画幅** | 8x10 大画幅相机（Calumet/Deardorff） |
| **镜头焦段** | 14英寸（约360mm等效），标准肖像镜头 |
| **光圈** | f/11 - f/22（大画幅标准工作光圈） |
| **快门** | 闪光灯同步速度或更长（大画幅快门较慢） |
| **ISO** | 大画幅胶片 ISO 64-160 |
| **布光** | 2-3 盏灯：主光 + 辅光 + 轮廓光 |

## 创作理念

- **"在按下快门之前，我已经看到了最终的影像"**——极致的前期准备和预判
- **深度研究拍摄对象**：阅读对象的自传、演讲、作品，理解其性格核心
- **"拍摄前我做的最重要的事是和这个人交谈"**——通过对话建立连接、观察表情
- **寻找"决定性瞬间"**：不是随机抓拍，而是在精心设计的框架中捕捉最真实的表情
- **"一张好的肖像，是能让你停下来注视良久的肖像"**
- **尊重每一位拍摄对象**：无论对方是总统还是平民，以同样的敬意对待

## 对标评价模板

> 这张肖像是否传递了拍摄对象的内在力量和性格？布光是否精确而有品质？是否有一个令人难忘的"签名"姿态？画面是否经得起长时间注视？

**评分维度**（1-10分）：
- 灵魂捕捉：是否展现了人物深层的性格特征
- 布光精度：光线是否精确服务于人物的表现
- 手势力量：手部/身体姿态是否增强了叙事
- 庄重感：画面是否传达了对象身份的分量
- 经典性：是否能经受时间考验成为时代肖像
