# 卡尔顿·沃特金斯（Carleton Watkins, 1829–1916）

## 一、生平简介

卡尔顿·沃特金斯是美国西部摄影的先驱，19世纪最伟大的风光摄影师之一。他是第一个系统性地拍摄约塞米蒂山谷的摄影师，其作品直接推动了1864年林肯总统签署《约塞米蒂法案》，使约塞米蒂成为美国受联邦保护的第一个自然风景区。沃特金斯以其巨幅湿版火棉胶（Wet Plate Collodion）摄影技术闻名，在极端野外条件下完成了令人叹为观止的拍摄。

- **1850年代**：在旧金山从肖像摄影学徒起步
- **1861年**：携带"巨型暗箱相机"（Mammoth Camera）和18×22英寸底片首次进入约塞米蒂
- **1860–1870年代**：系统拍摄加州风光、太平洋铁路建设、旧金山城市景观
- **1870年代**：因财务困境失去所有底片和设备，但仍坚持摄影
- **1906年旧金山大地震**：仅存的早期作品在仓库大火中大部分损毁
- **2005年**：作品《Pacific Coast, from Pigeon Point》拍出约3.9万美元高价

## 三、标志性技法

### 3.1 构图技法
- **全景式构图**：利用巨幅底片（18×22英寸）的宽广画幅，捕捉完整的山谷、瀑布全景
- **树木作为尺度参照**：在巨大的自然景观中加入树木、人物等作为尺度参照，传达景观的宏大尺度
- **重复与节奏**：利用树列、岩石排列创造视觉节奏
- **水平线构图**：广袤的地平线构图，展现西部的大与空旷

### 3.2 光影技法
- **自然光充分利用**：完全依赖自然光，在光线最佳时段拍摄
- **湿版曝光限制**：湿版火棉胶需要长曝光（数秒至数分钟），这决定了他的拍摄题材主要是静止景观
- **均匀照明偏好**：由于技术限制，偏好漫射光条件下拍摄，避免强烈的明暗对比

### 3.3 色调技法（棕褐色调/黑白）
- **湿版火棉胶独特质感**：银盐直接形成于玻璃板上，呈现独特的灰阶和细腻质感
- **棕褐色调（Sepia）**：后期常进行棕褐色调处理，赋予照片温暖的历史感
- **高信息密度**：巨幅底片记录了惊人的细节层次，放大后仍能看到远处的树叶和纹理
- **印刷蛋白相纸（Albumen Print）**：蛋白相纸工艺呈现的暖色调和细腻层次

### 3.4 技术参数参考
| 参数 | 典型设置 |
|------|----------|
| **相机** | "巨型暗箱相机"（Mammoth Camera），自制的18×22英寸画幅 |
| **底片** | 湿版火棉胶玻璃底片，18×22英寸（46×56cm） |
| **镜头** | 配备多组达盖尔式镜头，焦距涵盖广角到中焦 |
| **光圈** | f/8–f/16（受限于镜头工艺） |
| **快门速度** | 长曝光，数秒至数分钟 |
| **暗房** | 野外流动暗房（帐篷暗房），必须在底片湿的状态下完成曝光和显影 |
| **印刷** | 蛋白相纸接触印相（Albumen Print Contact Print） |

## 四、创作理念

### 4.1 核心哲学
> **"让东部的人看到西部的壮丽，用照片推动保护。"**

沃特金斯的摄影服务于双重目的：商业记录和艺术表达。他的约塞米蒂照片不仅是艺术品，更是推动自然保护的视觉证据。

### 4.2 技术先驱精神
- 面对极其艰苦的拍摄条件（骡队驮运数百磅设备、野外搭设暗房帐篷）
- 在无人尝试过的地方进行拍摄
- 持续改进设备和工艺，追求更大的画幅和更好的画质

### 4.3 纪实与艺术的融合
- 拍摄太平洋铁路建设过程，记录工业化进程
- 同时拍摄未被工业化触及的原始自然，形成对比
- 照片既是文献资料，也是独立的艺术作品

### 4.4 影响与遗产
- 他的照片直接影响了安塞尔·亚当斯的创作方向
- 推动了美国国家公园体系的建立
- 证明了摄影可以作为推动社会变革的视觉力量

## 七、学习路径建议

1. **历史研读**：阅读沃特金斯传记，了解19世纪美国西部摄影史
2. **湿版体验**：参加湿版火棉胶摄影工作坊，亲手体验这项古老工艺
3. **大画幅实践**：尝试4×5或8×10大画幅拍摄，体验构图严谨性
4. **纪实训练**：用摄影记录一个正在消逝的本地景观，体会沃特金斯的使命感
5. **历史比较**：对比沃特金斯和亚当斯拍摄的同一地点（如约塞米蒂），分析不同时代的摄影技术和审美差异
