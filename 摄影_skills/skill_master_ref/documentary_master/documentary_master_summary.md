## 五、学习优先级建议

### 5.1 初学者路径
1. **布列松** → 学习构图和瞬间的敏锐度
2. **麦凯瑞** → 学习色彩和人物肖像
3. **兰格** → 学习移情和人文关怀

### 5.2 进阶者路径
1. **史密斯** → 学习图片故事叙事
2. **埃文斯** → 学习"看"的智慧
3. **萨尔加多** → 学习长期项目和史诗级叙事

### 5.3 专业/研究者路径
1. **萨尔加多** → 深入学习长期项目方法论
2. **史密斯** → 学习图片故事完整工作流
3. **埃文斯** → 研究摄影与文化人类学的关系

## 六、风格交叉与融合

现代纪实摄影师可以将多位大师的风格元素融合：

- **布列松构图 + 麦凯瑞色彩** = 几何精准的彩色纪实
- **兰格移情 + 史密斯深度** = 深度人文图片故事
- **萨尔加多规模 + 埃文斯客观** = 大规模客观性纪实调查
- **布列松瞬间 + 史密斯叙事** = 充满动态张力的图片故事
- **麦凯瑞色彩 + 埃文斯构图** = 彩色的"美国表面"
