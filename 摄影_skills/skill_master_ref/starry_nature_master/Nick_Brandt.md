# Nick Brandt（尼克·布兰特）

## 生平简介

Nick Brandt 是英国著名的野生动物摄影艺术家，1964 年出生于伦敦。他早年从事音乐视频导演工作，曾为迈克尔·杰克逊、麦当娜等巨星执导 MV。1990 年代末期，Brandt 转向摄影，专注于非洲野生动物的记录。他的作品以黑白胶片摄影为主，风格深受古典绘画和早期摄影的影响。

Brandt 的摄影三部曲——《在地球上》（On This Earth, 2000）、《穿越伤痕的旅程》（A Shadow Falls, 2007）和《穿越裂开的土地》（Across the Ravaged Land, 2013）——是当代野生动物摄影的里程碑之作。后期，他转向更具观念性的创作，《继承尘埃》（Inherit the Dust, 2016）和《这一天》（This Empty World, 2019）将动物肖像与工业化的非洲景观并置，探讨人类活动对野生动物栖息地的侵蚀。

## 核心风格

- **黑白胶片美学**：坚持使用大画幅和中画幅黑白胶片，追求极致的影调层次
- **动物肖像画**：以拍摄人类肖像的方式拍摄动物，强调每只动物的个性和情感
- **低角度平视**：几乎总是以与动物平视的角度拍摄，消除人类的俯视优越感
- **极简背景**：非洲大地作为最简洁的背景，突出动物主体
- **社会议题融合**：后期作品将野生动物与工业化景观并置，强调环境危机

## 标志性技法

### 1. 大画幅/中画幅黑白胶片
- 使用 Pentax 67、Hasselblad 或 4×5 大画幅相机
- 黑白胶片（如 Ilford Delta、Kodak Tri-X）的丰富影调
- 刻意避免彩色，以消除视觉干扰，强化形式和情感
- 胶片的颗粒感和质感成为美学的一部分

### 2. 平视视角
- 总是尽可能低角度拍摄，与动物视线平齐
- 消除俯视带来的权力感，建立平等的情感连接
- 低角度同时强化了动物的宏伟和尊严
- 使用折叠式相机或低角度取景器实现这一视角

### 3. 长焦压缩与浅景深
- 使用 300mm 以上的长焦镜头（如 Pentax 67 的 400mm 镜头）
- 长焦压缩空间，将动物与背景紧密结合
- 大光圈创造浅景深，分离主体与背景
- 焦外成像（Bokeh）的柔和过渡增强绘画感

### 4. 自然光优先
- 几乎不使用闪光灯或人工光源
- 利用非洲黎明和黄昏的"金色时刻"
- 阴天柔光下的细腻影调
- 侧光和逆光的戏剧性效果

## 创作理念

> "当我拍摄一只大象时，我看到的不是一个'物种'或一个'标本'。我看到的是一个个体——一个有记忆、有情感、有社会关系、有独特性格的生命。我的工作是让你也看到这一点。"

Brandt 认为，野生动物摄影不应仅仅是"记录"，而应该是"肖像"——以同等的尊重和敏感对待动物。他的黑白风格选择不仅是美学偏好，更是一种道德立场：去除色彩的干扰，让观众直面动物的眼睛和灵魂。后期的环境主题作品则更进一步，他迫使观众面对一个事实：这些美丽的生灵所依赖的家园，正在人类的扩张中消失。

## 代表作品

- **《在地球上》系列**：非洲象、狮子、长颈鹿等动物的黑白肖像，展现一种宁静而庄严的美
- **《穿越伤痕的旅程》系列**：更近距离的动物特写，强调眼神和表情的情感深度
- **《穿越裂开的土地》系列**：加入非洲风景和人物，开始探讨人与动物的关系
- **《继承尘埃》系列**：将早期拍摄的动物肖像以巨幅尺寸打印，放置在工业化侵蚀的非洲 landscape 中，形成强烈对比
- **大象在乞力马扎罗山前**：Brandt 最著名的作品之一，大象剪影与非洲最高峰的壮丽合影
- **犀牛与管理员**：一只濒临灭绝的黑犀牛与其守护者的亲密肖像

## 学习要点

1. 黑白摄影不是彩色的"降级"，而是一种强调形式、影调和情感的独立语言
2. 平视视角是野生动物摄影中最能建立情感连接的拍摄角度
3. 长焦镜头不仅是"拍得更远"，更是压缩空间、创造绘画感的工具
4. 胶片摄影的局限（张数有限、即时反馈缺失）反而能迫使摄影师更专注、更深思熟虑
5. 野生动物摄影可以承载社会和环境议题，艺术性和 advocacy 并不矛盾
6. 耐心等待动物最放松、最真实的瞬间，比追逐动作更有价值
