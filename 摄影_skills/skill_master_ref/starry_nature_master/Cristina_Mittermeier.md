# Cristina Mittermeier（克里斯蒂娜·米特迈尔）

## 生平简介

Cristina Mittermeier 是墨西哥海洋生态摄影师、作家和保护活动家。她出生于墨西哥城，在尤卡坦半岛附近的沿海小镇长大，从小就与海洋建立了深厚的情感联系。她拥有生物化学学位，后在富士胶片和国家地理的资助下转向摄影，成为"保护摄影"（Conservation Photography）这一领域的开创者和最重要的倡导者之一。

2014 年，Mittermeier 与 Paul Nicklen 共同创立了 SeaLegacy 基金会，致力于通过影像推动全球海洋保护。她定义了"保护摄影"的概念——一种以摄影为工具，直接服务于环境保护和生物多样性保护的视觉叙事形式。她的足迹遍布全球海洋，从北极到南极，从大堡礁到加拉帕戈斯。

## 核心风格

- **保护摄影的定义者**：将摄影从单纯的记录提升为保护行动的工具
- **人与海洋的共生叙事**：不仅拍摄海洋生物，更关注沿海社区与海洋的关系
- **水下广角环境摄影**：强调海洋生物与其栖息地的整体关系，而非孤立的个体
- **原住民视角**：尊重和呈现原住民社区的传统海洋知识和保护实践

## 标志性技法

### 1. 水下广角环境摄影
- 使用 16-35mm 水下广角镜头，近距离拍摄大型海洋生物
- 强调动物与其环境的关系——珊瑚礁、海草床、 kelp forest
- 利用自然光（阳光穿透水面）创造戏剧性的水下光束
- 避免过度使用闪光灯，保持水下场景的自然氛围

### 2. 水上-水下分割构图（Over-Under）
- 使用大口径球罩（Dome Port）拍摄水面分割画面
- 上半部分展现天空、云层、山脉或人类活动
- 下半部分展现水下生态系统
- 一个画面同时讲述两个世界的故事

### 3. 原住民社区的长期跟拍
- 在一个社区长期居住，建立信任和友谊
- 记录传统的捕鱼方式、仪式和海洋管理方式
- 将人物肖像与海洋环境结合，展现文化多样性
- 让被摄者参与叙事，而非仅仅作为"被拍摄的对象"

### 4. 无人机航拍海洋
- 使用无人机展现海洋生态系统的宏观面貌
- 记录珊瑚礁白化、海岸线变化、海洋污染等环境问题
- 鲸鱼群、鲨鱼群、蝠鲼群的大规模聚集航拍
- 将航拍与地面/水下拍摄结合，形成多维度叙事

## 创作理念

> "海洋不仅是生物的栖息地，更是人类文明的摇篮。当你看到一个小岛上的人们如何依赖珊瑚礁生存，你就会明白：保护海洋不是在保护某种遥远的'自然'，而是在保护我们自己。"

Mittermeier 认为，最有效的保护摄影不是展示破坏和绝望，而是展示人与自然的和谐共存。她致力于寻找那些还在"正确运作"的海洋生态系统，以及那些正在努力保护海洋的社区。她的信念是：让人们看到希望，他们才会行动。

## 代表作品

- **北极因纽特人与海冰**：记录加拿大北极地区因纽特社区与融化海冰的关系，气候变化的直接见证
- **大堡礁白化**：澳大利亚大堡礁珊瑚白化的震撼影像，推动全球珊瑚保护关注
- **墨西哥尤卡坦洞潜**：拍摄墨西哥 cenotes（天然井）中的水下生态系统，展现淡水与海洋的隐秘连接
- **虎鲨与潜水员**：巴哈马水域中虎鲨与潜水员的和平共处，改变人们对鲨鱼的恐惧认知
- **座头鲸母子**：汤加海域中座头鲸母亲与幼崽的亲密互动，海洋哺乳动物的情感世界
- **海达族与鲑鱼**：加拿大西海岸原住民海达族的传统鲑鱼捕捞，文化与生态的交织

## 学习要点

1. 保护摄影的核心是"行动导向"——每张照片都应该服务于一个保护目标
2. 水下广角近距离拍摄是大型海洋生物摄影的最佳方式，强调动物与环境的关系
3. 水面分割构图（Over-Under）是海洋摄影中最具叙事力的技巧之一
4. 与原住民和社区建立信任需要时间和尊重，急功近利的拍摄无法获得真实的故事
5. 无人机航拍为海洋摄影提供了前所未有的宏观视角，但需要注意不要干扰动物
6. 展示希望比展示破坏更能激发保护行动——寻找积极的故事和解决方案
