# 风格推荐逻辑

## 一、概述

本文件定义了根据用户照片的风格特征向量，匹配最接近的大师风格，并给出个性化风格发展建议的逻辑。

### 2.2 大师CLIP特征向量来源
- 从每位大师的10-20张代表作品中提取CLIP图像嵌入向量
- 对所有代表作品的向量取平均值，得到该大师的"风格核心向量"
- 存储在向量数据库中，支持快速相似度检索

## 三、匹配算法

### 3.1 两阶段匹配策略

#### 阶段一：CLIP向量粗筛
```
1. 提取用户照片的CLIP图像嵌入向量（512维，L2归一化）
2. 计算与所有大师向量的余弦相似度
3. 返回Top-5候选大师列表
4. 阈值过滤：相似度 > 0.3 才纳入候选
```

#### 阶段二：特征向量精排
```
1. 对Top-5候选大师进行详细特征比对
2. 计算加权欧氏距离：

   distance = Σ(weight_i × |user_feature_i - master_feature_i|)

   权重分配：
   - color: 0.25
   - tone: 0.25
   - composition: 0.25
   - light: 0.15
   - subject: 0.10

3. 距离越小匹配度越高
4. 返回Top-3匹配结果，附带匹配度百分比
```

### 3.2 匹配度计算公式

```
match_score = (1 - normalized_distance) × 100

其中：
normalized_distance = weighted_distance / max_possible_distance

max_possible_distance = Σ(weight_i × feature_range_i)

匹配度分级：
- 90–100%：高度匹配（照片风格与大师非常接近）
- 70–89%：中度匹配（有明显相似特征）
- 50–69%：低度匹配（有部分相似元素）
- <50%：风格差异较大（可能属于混合风格或独特风格）
```

### 3.3 混合风格检测

```
当Top-2大师的匹配度差距 < 10%时，判定为混合风格：

mixed_style = {
    "primary": "大师A",
    "secondary": "大师B",
    "mix_ratio": "A: B的匹配比例",
    "description": "你的作品融合了大师A的[特征]和大师B的[特征]"
}
```

## 四、风格发展建议生成

### 4.1 基于匹配结果的建议

```
IF 匹配度 >= 80%:
    建议 = "你的风格已非常接近{大师名}的成熟风格"
    建议 += "下一步可以尝试：
           1. 研究{大师名}的非典型作品，拓展风格边界
           2. 尝试将{大师名}的技法与其他大师的构图/色彩融合
           3. 开创个人风格变体——在{大师名}基础上加入你的独特元素"

IF 60% <= 匹配度 < 80%:
    建议 = "你的作品有{大师名}的影子，但还有发展空间"
    建议 += "具体改进方向：
           1. [基于差距分析的具体建议]
           2. [针对弱项维度的练习建议]
           3. [推荐研读的大师作品/书籍]"

IF 40% <= 匹配度 < 60%:
    建议 = "你的作品有与{大师名}相似的某些元素"
    建议 += "你可以考虑：
           1. 是否有意朝这个方向发展？
           2. [针对性技法训练建议]"

IF 匹配度 < 40%:
    建议 = "你的风格比较独特，不完全匹配已有大师风格"
    建议 += "这可能是好事——你在形成个人风格"
    建议 += "可以从多位大师中各取所长：
           1. [最接近的3位大师及可借鉴的方面]"
```

## 六、特殊情况处理

### 6.1 多人合照
- 当检测到多人合照时，降低"单个大师"匹配权重
- 优先匹配布列松（街头群像）、萨尔加多（密集人群）或埃文斯

### 6.2 明显修图痕迹
- 当检测到过度磨皮、过度液化等痕迹时
- 不进行大师匹配，转而给出修图建议（移交retouch_agent）
- 附注："检测到明显后期处理，建议参考修图指南先调整"

### 6.3 新手/技术问题照片
- 当检测到严重的技术问题（曝光错误、对焦失败、水平线倾斜等）
- 先进行技术问题诊断
- 再进行风格匹配（基于"潜在风格"而非当前技术表现）

### 6.4 创意图/合成图
- 当检测到明显的合成/创意处理时
- 标注为"创意/合成摄影"
- 匹配Cindy Sherman（观念摄影）或Gursky（数字合成）等创意类大师
