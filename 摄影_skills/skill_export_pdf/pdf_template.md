```

├── 封面（Cover Page）

├── 第一章：原图分析区
│   ├── 原图展示
│   ├── EXIF信息摘要

├── 第二章：构图分析区
│   ├── 九宫格叠加分析图
│   ├── 构图线标注图

│   └── 构图改进建议示意图

└── 封底（Back Cover）
```

## 四、第二章：构图分析区

### 4.1 九宫格叠加分析图
```
在原图上叠加九宫格（三分法网格）线：
- 4条等距水平线和垂直线
- 将画面分为9个等面积区域
- 在4个交叉点标注"交叉点"
- 主体位置与最近交叉点的距离标注

标注颜色：半透明白色线 + 红色交叉点
```

### 4.2 构图线标注图
```
在原图上标注构图要素：
- 引导线：黄色实线 + 箭头标注方向
- 水平线：绿色虚线（如倾斜则标注偏差角度）
- 对称轴：蓝色虚线
- 前景/中景/远景分割线：白色虚线
- 负空间区域：浅灰色半透明填充
- 框架构图：橙色虚线框出框架元素
```

### 4.3 构图评价文字模板
```
【构图评分】75/100

【优点】
1. 主体位置合理，接近右侧三分线
2. 前景引导线有效，将视线引向主体
3. 画面有明确的前景-中景-远景层次

【问题】
1. 水平线向右倾斜约1.5度（红色标注）
2. 主体偏右过度，左侧留白过多
3. 顶部空间略显紧凑

【改进建议】
1. 校正水平线（旋转-1.5度）
2. 裁剪左侧部分，使主体更居中
3. 增加顶部空间（上移画面或改变拍摄角度）
```

## 八、PDF输出规格

- 分辨率：150 DPI（屏幕阅读）/ 300 DPI（打印）
- 文件格式：PDF/A（长期存档）
- 文件大小控制：< 10MB
- 色彩空间：sRGB
- 页面编号：右下角
- 页眉：报告标题（浅灰色，小字号）
- 页脚：页码 + "Photo Agent 生成"
