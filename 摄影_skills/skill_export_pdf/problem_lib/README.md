# 问题图库说明

## 一、概述

本目录用于存储常见摄影问题的示例图片和分类标准。问题图库用于：
- 对比用户照片中的问题
- 生成问题诊断报告
- 提供视觉化的问题示例

## 二、问题分类体系

### 2.1 曝光问题

| 问题代码 | 问题名称 | 严重程度 | 检测方法 | 典型表现 |
|----------|----------|----------|----------|----------|
| EXP-001 | 过曝 | 高 | 直方图高光溢出 | 天空纯白、皮肤过亮、高光区域无细节 |
| EXP-002 | 欠曝 | 高 | 直方图暗部裁切 | 画面过暗、暗部无细节 |
| EXP-003 | 高光溢出 | 中 | 直方图右侧裁切 | 天空、光源等高光区域完全白色 |
| EXP-004 | 暗部缺失 | 中 | 直方图左侧裁切 | 暗部纯黑，无任何纹理 |
| EXP-005 | 曝光不均 | 中 | 局部亮度分析 | 画面一侧过亮一侧过暗 |
| EXP-006 | 闪光过曝 | 中 | 人脸高光检测 | 人脸过亮，与环境光不协调 |

**示例文件命名**：
- `exp_001_overexposed_01.jpg`
- `exp_002_underexposed_01.jpg`

### 2.2 构图问题

| 问题代码 | 问题名称 | 严重程度 | 检测方法 | 典型表现 |
|----------|----------|----------|----------|----------|
| CMP-001 | 水平线倾斜 | 中-高 | 霍夫线变换检测 | 地平线/水面不水平，倾斜>1度 |
| CMP-002 | 主体偏移 | 中 | 显著区域检测 | 主体偏离视觉重心 |
| CMP-003 | 画面切割 | 中 | 边缘检测 | 人物/物体被画面边缘不自然切割 |
| CMP-004 | 前景杂乱 | 中-低 | 前景区域复杂度分析 | 前景元素过多分散注意力 |
| CMP-005 | 留白不足 | 低 | 负空间计算 | 画面过于拥挤，无呼吸空间 |
| CMP-006 | 留白过多 | 低 | 主体面积占比 | 主体过小，画面空旷 |
| CMP-007 | 干扰物 | 中 | 显著区域检测 | 画面中存在与主题无关的干扰元素 |
| CMP-008 | 地平线居中 | 低 | 水平线位置分析 | 水平线正好在画面中央（通常不佳） |
| CMP-009 | 透视畸变 | 中 | 垂直线检测 | 建筑等垂直线倾斜（非大广角创意效果） |

**示例文件命名**：
- `cmp_001_tilted_horizon_01.jpg`
- `cmp_009_distortion_01.jpg`

### 2.3 对焦/锐度问题

| 问题代码 | 问题名称 | 严重程度 | 检测方法 | 典型表现 |
|----------|----------|----------|----------|----------|
| FOC-001 | 对焦失败 | 高 | 拉普拉斯方差分析 | 主体模糊，完全失焦 |
| FOC-002 | 对焦偏移 | 中 | 局部清晰度分析 | 焦点落在错误位置（如背景清晰，主体模糊） |
| FOC-003 | 运动模糊 | 中 | 方向性模糊检测 | 主体或相机运动导致的模糊 |
| FOC-004 | 景深不足 | 低 | 景深范围计算 | 需要全景深但光圈过大导致部分区域模糊 |
| FOC-005 | 双重曝光残留 | 中 | 像素异常检测 | 快门故障导致的叠影 |
| FOC-006 | 手抖模糊 | 中 | 全局模糊分析 | 快门速度过慢导致的全局轻微模糊 |

**示例文件命名**：
- `foc_001_out_of_focus_01.jpg`
- `foc_003_motion_blur_01.jpg`

### 2.4 色彩问题

| 问题代码 | 问题名称 | 严重程度 | 检测方法 | 典型表现 |
|----------|----------|----------|----------|----------|
| CLR-001 | 白平衡偏色 | 中 | 色温估算 | 整体偏蓝/偏黄/偏绿 |
| CLR-002 | 色彩过饱和 | 中-低 | HSL分析 | 色彩过于浓烈，不自然 |
| CLR-003 | 色彩不饱和 | 低 | 全局饱和度分析 | 色彩灰暗平淡 |
| CLR-004 | 色彩偏移 | 中 | 各通道分析 | 特定颜色通道异常（如紫色偏移） |
| CLR-005 | 混合光源色温 | 中 | 局部色温分析 | 不同区域色温不一致（如室内窗户+灯光） |
| CLR-006 | 色差/紫边 | 低 | 边缘色彩异常 | 高对比度边缘出现紫色/绿色色边 |
| CLR-007 | 噪点/彩色噪点 | 中-低 | 噪点分析 | 高ISO导致的全局或局部噪点 |
| CLR-008 | 逆光色彩偏暖 | 低 | 色温分析 | 逆光导致整体偏暖/偏橙 |

**示例文件命名**：
- `clr_001_wb_shift_01.jpg`
- `clr_006_chromatic_aberration_01.jpg`

## 三、问题严重程度定义

| 等级 | 标识 | 定义 | 处理建议 |
|------|------|------|----------|
| **严重** | 红色 | 照片的核心问题，严重影响观感 | 必须修正，优先级最高 |
| **中等** | 橙色 | 明显的问题但可以接受 | 建议修正，优先级中等 |
| **轻微** | 黄色 | 细微的问题，不影响整体 | 可选修正，优先级低 |
| **无问题** | 绿色 | 该维度表现良好 | 无需修正 |

## 四、问题检测算法映射

```
每个问题对应的检测算法：

曝光问题：
- EXP-001/002 → 直方图分析
- EXP-003/004 → 直方图裁切检测
- EXP-005 → 图像分块亮度计算
- EXP-006 → 人脸检测+局部亮度分析

构图问题：
- CMP-001 → OpenCV霍夫线变换
- CMP-002 → 显著区域检测+重心计算
- CMP-003 → 边缘切割检测
- CMP-004 → 前景区域复杂度分析
- CMP-005/006 → 负空间计算
- CMP-007 → 显著区域异常检测
- CMP-008 → 水平线位置分析
- CMP-009 → 垂直线角度分析

对焦问题：
- FOC-001/002 → 拉普拉斯方差+局部清晰度热力图
- FOC-003 → 方向性模糊检测（Radon变换）
- FOC-004 → EXIF光圈信息+景深计算
- FOC-006 → 全局模糊检测

色彩问题：
- CLR-001 → 灰场估算+色温估算
- CLR-002/003 → 全局饱和度统计
- CLR-004 → RGB通道独立分析
- CLR-005 → 局部色温差异计算
- CLR-006 → 边缘色彩异常检测
- CLR-007 → 空间频率分析（噪点检测）
```
