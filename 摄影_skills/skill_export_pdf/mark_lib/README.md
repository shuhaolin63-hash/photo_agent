# 标注图库说明

## 一、概述

本目录用于存储构图分析中使用的标注元素素材。这些素材将叠加在用户照片上，形成构图分析图。

## 二、标注类型与规范

### 2.1 构图线标注

#### 三分法网格（九宫格）
```
文件格式：PNG（透明背景）
尺寸：适配多种比例（3:2、4:3、16:9、1:1）
线条规格：
- 颜色：白色，opacity 60%
- 线宽：1px
- 网格：2横2纵，将画面分为9等份
- 交叉点标记：红色圆点，直径6px

命名规则：
- grid9_3x2.png（3:2比例）
- grid9_4x3.png（4:3比例）
- grid9_16x9.png（16:9比例）
- grid9_1x1.png（1:1比例）
```

#### 黄金螺旋（斐波那契螺旋线）
```
文件格式：SVG（可缩放矢量）
颜色：白色，opacity 50%
线条宽度：1.5px
螺旋方向：顺时针/逆时针各一套
黄金比例点：红色圆点标注

命名规则：
- golden_spiral_cw.svg（顺时针）
- golden_spiral_ccw.svg（逆时针）
```

#### 对角线
```
文件格式：PNG（透明背景）
颜色：黄色，opacity 60%
线条宽度：1px，虚线样式
类型：左上→右下、右上→左下

命名规则：
- diagonal_lr.png（左上→右下）
- diagonal_rl.png（右上→左下）
- diagonal_cross.png（X形交叉对角线）
```

#### 水平线标注
```
文件格式：PNG（透明背景）
颜色：绿色（校正后）/ 红色（倾斜）
线条宽度：1.5px
附带：倾斜角度数值标注（如"-1.5°"）

命名规则：
- horizon_correct.png（绿色，水平）
- horizon_tilt.png（红色，倾斜，角度可变）
```

#### S形曲线标注
```
文件格式：PNG（透明背景）
颜色：白色，opacity 60%
线条宽度：1.5px，虚线样式
箭头：标注S形走向

命名规则：
- s_curve_default.png
```

#### 引导线箭头
```
文件格式：PNG（透明背景）
颜色：黄色，opacity 70%
线条宽度：1.5px
箭头：末端标注方向
类型：单向箭头、双向箭头

命名规则：
- arrow_single.png
- arrow_double.png
- arrow_set_radial.png（放射状引导线组）
```

### 2.2 框架构图标注
```
文件格式：PNG（透明背景）
颜色：橙色，opacity 50%
线条宽度：1.5px，虚线样式
用途：标注门框、窗框、拱门等自然框架

命名规则：
- frame_rect.png（矩形框架）
- frame_arch.png（拱形框架）
- frame_custom.png（自定义形状框架）
```

### 2.3 层次分割线
```
文件格式：PNG（透明背景）
颜色：白色，opacity 40%
线条宽度：1px，虚线样式
用途：标注前景/中景/远景的分割位置

命名规则：
- layer_fg_mid.png（前景-中景分割线）
- layer_mid_bg.png（中景-远景分割线）
- layer_all.png（全部三层分割线）
```

### 2.4 区域填充标注
```
文件格式：PNG（透明背景）
用途：标注负空间/正空间区域
颜色：浅灰色，opacity 20%（负空间填充）
      浅蓝色，opacity 15%（主体区域高亮）

命名规则：
- negative_space_fill.png
- subject_highlight.png
```

## 三、使用规范

### 3.1 叠加规则
- 标注层始终位于照片之上
- 标注线条不要遮挡照片主体
- 交叉点/关键点标注使用醒目颜色（红色/黄色）
- 辅助线使用低调颜色（白色半透明）
- 倾斜/问题标注使用警告色（红色/橙色）

### 3.2 尺寸适配
- 所有PNG标注素材应提供多种尺寸版本（适配不同分辨率）
- SVG素材可自由缩放，无需多尺寸
- 叠加前应根据照片实际尺寸调整标注素材大小

### 3.3 输出要求
- 标注后的图片应同时提供"带标注版"和"无标注版"
- 带标注版用于PDF报告中的"构图分析区"
- 无标注版即原图，用于对比参考

## 四、标注组合方案

| 分析类型 | 使用的标注组合 |
|----------|---------------|
| 三分法分析 | 九宫格 + 交叉点 + 主体位置标记 |
| 黄金比例分析 | 黄金螺旋 + 黄金比例点 + 主体位置标记 |
| 引导线分析 | 引导线箭头 + 层次分割线 + 主体标记 |
| 水平线分析 | 水平线标注 + 角度数值 |
| 对称分析 | 对称轴线 + 两侧区域高亮 |
| 负空间分析 | 负空间区域填充 + 主体区域高亮 |
| 框架分析 | 框架轮廓线 + 主体标记 |
| 综合分析 | 九宫格 + 引导线 + 水平线 + 主体标记 |
