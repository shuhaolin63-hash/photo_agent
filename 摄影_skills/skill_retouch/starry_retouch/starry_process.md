# 星空修图

## 一、降噪

### 1.1 星空摄影的噪点挑战
- 星空摄影必然使用高ISO（1600–12800）
- 长曝光也会产生热噪点（特别是夏季和长时间曝光）
- 降噪是星空修图的第一步，也是最关键的一步

### 1.2 单帧降噪方案

**Lightroom降噪：**
```
参数（ISO 1600-3200）：
- Luminance: 40-60
- Detail: 30-40
- Contrast: 10-20

参数（ISO 3200-6400）：
- Luminance: 60-80
- Detail: 25-35
- Contrast: 10-15

参数（ISO 6400+）：
- Luminance: 80-100
- Detail: 20-30
- Contrast: 5-10

Color降噪：
- Color: 30-50
- Smoothness: 50

注意事项：
- 降噪会损失星星的细节，需要在噪点和星星之间平衡
- Detail参数不要太低，否则星星会变成模糊的光斑
- 对于银河核心区域，可以适当降低降噪强度以保留银河结构
```

**DxO DeepPRIME降噪（推荐专业方案）：**
```
- DeepPRIME/DeepPRIME XD是当前最佳的AI降噪方案
- 在保留星空细节的同时有效去除噪点
- 优于Lightroom的传统降噪算法
- 推荐用于最终输出的降噪步骤
```

**Adobe AI降噪（Lightroom 2023+）：**
```
路径：Detail面板 → Noise Reduction → AI降噪
- 适合高ISO星空照片
- 保留星星细节的能力优于传统算法
- 推荐强度：70-100%
```

### 1.3 拍摄降噪技巧（前期）
```
- 关闭长时间曝光降噪（相机内置NR）
- 使用ISO invariance策略：使用最低原生ISO，后期提亮
- 拍摄暗场帧（Dark Frame）：盖上镜头盖，同参数拍摄1-5张
- 拍摄偏置帧（Bias Frame）：最短曝光+最低ISO
- 拍摄平场帧（Flat Frame）：使用白板/天光，均匀曝光
```

## 二、叠加（Stacking）

### 2.1 什么是叠加
- 叠加是将多张同一场景的照片合成一张，以提高信噪比的技术
- 理论上，N张照片叠加后信噪比提升sqrt(N)倍
- 例如：16张叠加后信噪比提升4倍（等效ISO降低4倍）

### 2.2 叠加工作流

**方法一：Median叠加（最简单）**
```
适用场景：固定机位拍摄的星空
步骤：
1. 将所有帧导入Photoshop作为图层
2. 全选图层 → Edit → Auto-Align Layers
3. 全选图层 → Layer → Smart Object → Convert to Smart Object
4. Layer → Smart Objects → Stack Mode → Median
5. Median模式会自动去除飞机轨迹、卫星等随机噪点
```

**方法二：Starry Landscape Stacker（Mac专用）**
```
这是星空摄影专用的叠加软件
步骤：
1. 导入所有帧
2. 设置天空阈值（区分天空和地面）
3. 选择合成模式（Median/Max/ Mean）
4. 选择对齐参考帧
5. 执行合成
6. 导出结果
```

**方法三：Sequator / StarryAutoStacker（Windows）**
```
Windows平台可用的星空叠加工具
原理和步骤与Starry Landscape Stacker类似
```

**方法四：DeepSkyStacker（深空摄影专用）**
```
适用场景：望远镜拍摄的深空天体
步骤：
1. 注册（Register）所有帧 → 对齐
2. 校准（Calibrate）→ 暗场/平场/偏置场校正
3. 叠加（Stack）→ 合成
4. 输出32位TIFF文件
5. 后期导入Photoshop/Lightroom处理
```

### 2.3 叠加参数参考
```
帧数建议：
- 入门：8-16帧（明显改善）
- 进阶：32-64帧（显著改善）
- 专业：100+帧（极限信噪比）

每帧参数：
- ISO：根据目标亮度调整（通常1600-6400）
- 曝光时间：15-25秒（避免星轨，遵守500法则）
- 间隔：间隔2-5秒（减少传感器过热）
- 总帧数 = 总拍摄时间 / 单帧时间

500法则：
最大曝光时间（秒）= 500 / 焦距（mm）
例：14mm → 500/14 = 35秒
例：24mm → 500/24 = 20秒
例：50mm → 500/50 = 10秒
```

## 三、星轨合成

### 3.1 星轨拍摄参数
```
拍摄方法：连续拍摄数百张照片，后期合成
参数：
- 曝光时间：30秒（较长以获取更多星轨长度）
- ISO：800-1600
- 间隔：1-2秒
- 总帧数：200-500张（取决于期望的星轨长度和拍摄时长）
- 总时长：约2-4小时
- 对焦：手动对焦至无限远，然后微调
- 防抖：关闭
- 文件格式：RAW（单帧也需要调整）或JPEG（合成后不再调）
```

### 3.2 StarStaX合成
```
步骤：
1. 导入所有帧
2. 设置合成模式：
   - Lighten（叠加模式）：每帧取最亮的像素
   - 是最常用的星轨合成模式
3. 可选：设置间隙填充（Gap Filling）
4. 可选：设置前景合成方式（使用第一帧或最后一帧的前景）
5. 执行合成
6. 导出结果
```

### 3.3 Photoshop合成
```
步骤：
1. 导入所有帧作为图层
2. 全选 → 混合模式改为"Lighten"（变亮）
3. 所有星轨会自动叠加
4. 使用蒙版保留最佳前景帧
5. 合并图层
6. 后续调色和锐化
```

### 3.4 单帧星轨（长曝光法）
```
方法：使用B门，单次曝光数小时
参数：
- B门（或间隔拍摄后合成）
- ISO：100-400（低ISO减少噪点）
- 曝光时间：1-4小时
- 光圈：f/4-f/8
- 附件：充足的电池、快门线、防雾加热带

优点：方法简单
缺点：电池消耗大，一旦失败需重新拍摄
推荐：先用连续拍摄法（更安全），再尝试单帧法
```

## 四、色彩增强

### 4.1 星空色彩基础
```
星空中的色彩来源：
- 银河核心：暖黄色/金色（密集恒星区域）
- 星云/发射区：红色/粉色（氢Alpha发射）
- 反射星云：蓝色（反射恒星光线）
- 天空背景：深蓝/深紫色（大气散射）
- 星星颜色：蓝白（高温星）、黄色（类太阳星）、红色（低温星）
- 光污染：橙黄色/粉红色（钠灯/LED灯）
```

### 4.2 Lightroom星空调色方案
```
步骤1：基础调整
- 曝光：+0.5~+1.5（提亮星空）
- 对比度：+10~+20
- 高光：-20~-40（保护亮星不过曝）
- 阴影：+50~+80（提亮银河结构）
- 清晰度：+15~+30（增强银河纹理）
- Dehaze：+15~+30（增强银河对比）
- 饱和度：+5~+10

步骤2：色彩调整
- 色温：-10~-30（天空偏蓝）
- 色调：0~+10
- HSL面板：
  - Blue Saturation: +10~+20
  - Purple Saturation: +10~+15
  - Orange/Yellow Saturation: +5~+10（恒星颜色）
  - Red Saturation: +5~+15（星云区域）

步骤3：分离色调
- 高光：浅蓝色/浅紫色（H:210-270, S:10-15）
- 阴影：深蓝色（H:210, S:5-10）

步骤4：局部增强
- 径向滤镜覆盖银河核心：
  - 曝光：+0.3~+0.5
  - 饱和度：+10~+15
  - 清晰度：+10~+15
```

### 4.3 星空与前景的平衡处理
```
问题：星空和前景的曝光需求不同
- 星空需要高ISO+长曝光
- 前景如果曝光过长会过曝或产生运动模糊

解决方案一：双曝光合成
- 拍摄一张星空对焦点无限远的照片
- 拍摄一张前景对焦近处+低ISO+适当曝光的照片
- 后期在Photoshop中合成

解决方案二：前景光绘
- 拍摄星空时用手电筒短暂照亮前景
- 多张叠加中利用Median模式去除手电光

解决方案三：前景蓝调（Blue Hour融合）
- 在蓝调时刻（日落后30-60分钟）拍摄前景
- 星空拍摄在完全天黑后
- 后期合成天空和前景
```

### 4.4 常见星空修图问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| **星星变成光斑** | 降噪过度或锐化过度 | 降低降噪强度，降低锐化量 |
| **银河细节丢失** | 对比度不足 | Dehaze + Clarity叠加使用 |
| **色彩不自然** | 色彩增强过度 | 降低HSL调整幅度，参考真实星空色彩 |
| **光污染严重** | 城市附近拍摄 | 前期选择暗天地点；后期用渐变滤镜压暗地平线 |
| **前景模糊** | 超焦距计算错误 | 使用双曝光合成或前景对焦后合成 |
