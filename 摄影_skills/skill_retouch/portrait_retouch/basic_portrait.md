# 人像基础修图

## 一、磨皮

### 1.1 原则
- 保留皮肤纹理（毛孔、细纹），去除不均匀色斑
- 磨皮的目标是"均匀+自然"，而非"无纹理的塑料感"
- 不同用途的磨皮程度不同：社交媒体可适度，商业人像需精细

### 1.2 Lightroom磨皮方案

**方法一：纹理面板+ clarity调整**
```
步骤：
1. 使用蒙版（面部检测/颜色范围）选中皮肤区域
2. Texture 参数 -10 ~ -30（去除粗糙纹理）
3. Clarity 参数 -15 ~ -30（柔化皮肤）
4. Sharpness 参数 +20 ~ +40（保留轮廓清晰度）
5. 调整蒙版羽化和边缘，确保不影响眼睛/嘴唇/头发
```

**方法二：范围蒙版精确控制**
```
步骤：
1. 创建渐变或画笔蒙版
2. 设置颜色范围蒙版，取样皮肤颜色
3. 仅对皮肤区域应用上述参数
4. 使用"显示蒙版"检查覆盖范围
```

### 1.3 Photoshop磨皮方案

**方法一：频率分离（Frequency Separation）**
```
步骤：
1. 复制背景图层两次，得到"高频层"和"低频层"
2. 低频层：高斯模糊 4-8px → Image > Apply Image，选择低频层
3. 高频层：Image > Apply Image，用低频层做减法
4. 低频层用套索+模糊去除色斑（不影响纹理）
5. 高频层用修复画笔去除痘痘/疤痕（不影响色彩）
6. 合并图层

参数参考：
- 高斯模糊半径：3-8px（根据皮肤特写程度调整）
- 低频层处理：大块色斑用模糊，小面积色差用画笔
- 高频层处理：修复画笔+仿制图章，逐点修复
```

**方法二：D&B修图（Dodge & Burn）**
```
原理：用减淡和加深工具（或曲线调整图层+黑白蒙版）来均匀皮肤明暗

步骤：
1. 创建曲线调整图层，向上提亮（减淡/亮化层）
2. 添加黑色蒙版
3. 用白色柔边画笔在暗斑区域涂抹
4. 创建第二个曲线调整图层，向下压暗（加深层）
5. 添加黑色蒙版
6. 用白色柔边画笔在过亮区域涂抹
7. 使用极低的不透明度（1-5%），反复涂抹

优点：
- 完全保留皮肤纹理
- 效果最自然
- 适合商业人像

缺点：
- 耗时较长
- 需要熟练的绘画技能
```

### 1.4 醒图磨皮方案
```
路径：修复 → 美肤
推荐参数：
- 美肤程度：30-50（不要拉满）
- 细节保留：开启
- 均匀肤色：开启，程度20-40
```

### 1.5 磨皮常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| **塑料感** | 磨皮过度，纹理全失 | 降低磨皮强度，保留texture |
| **皮肤颜色不均** | 频率分离低频层处理不彻底 | 检查低频层，确保色块均匀 |
| **五官模糊** | 蒙版覆盖了五官区域 | 精确蒙版，排除眼睛/嘴唇/眉毛 |
| **边缘白边** | 频率分离参数不当 | 检查Apply Image参数 |

## 二、液化

### 2.1 原则
- **微调为主**，幅度控制在5-10%以内
- 保持身体比例的自然合理
- 避免破坏背景线条（水平线、建筑边缘等）
- 遵循骨骼结构，不要"创造"不可能的体态

### 2.2 常用液化操作

**Lightroom：**
```
路径：Develop → 纠正面部特征
工具：
- 面部大小（Face Size）：微调面部大小
- 下颌（Jaw）：微调下巴轮廓
- 眉毛高度（Brow Height）：微调眉眼距离
- 鼻子（Nose Width）：微调鼻翼宽度
- 嘴（Mouth Width/Height）：微调嘴部比例
- 眼睛（Eye Size）：微调眼睛大小
```

**Photoshop液化滤镜：**
```
路径：Filter > Liquify
工具：
- 向前变形工具（Forward Warp Tool）：推挤调整轮廓
  - 画笔大小：中等
  - 画笔压力：20-40（低压力多次微调）
- 褶皱工具（Pucker Tool）：收窄区域（腰、臂）
- 膨胀工具（Bloat Tool）：放大区域（眼睛、胸部）
- 冻结蒙版（Freeze Mask Tool）：保护不需要变形的区域
```

### 2.3 液化禁忌
- 不改变头部大小比例（大头畸形）
- 不制造不自然的腰臀比
- 不改变背景中的水平线/建筑线条
- 不在人物边缘留下明显的扭曲痕迹

## 三、美白

### 3.1 原则
- "美白"本质是肤色均匀和亮度提升，不是"漂白"
- 保留皮肤的自然暖色调，避免发灰/发青
- 美白应针对暗沉区域，而非全脸 uniformly 提高

### 3.2 操作方法

**Lightroom肤色校正：**
```
方法一：HSL面板
- Luminance → Orange: +10 ~ +30（提亮肤色通道）

方法二：曲线+蒙版
- RGB曲线微微向上提升整体亮度
- 创建蒙版，仅选中皮肤区域
- 适度提亮

方法三：校准面板
- Blue Primary Hue: 向左移动 → 肤色偏暖
- Red Primary Saturation: 微调 → 增加肤色饱和度
```

**Photoshop肤色校正：**
```
方法一：曲线+蒙版
- 创建曲线调整图层
- 向上提亮中间调
- 用颜色范围蒙版选中皮肤
- 羽化蒙版边缘

方法二：可选颜色
- Selective Color → Red/Yellow
- 适度调整Yellow的Black通道（减少黄色中的黑色成分）
```

**醒图方案：**
```
路径：调整 → 色温/色调/光感
- 色温：+10~+20（偏暖美白，不灰白）
- 色调：+5~+15（微红润）
- 光感：+10~+20（提亮）
- 局部提亮：对人脸做局部提亮蒙版
```

## 四、眼睛提亮

### 4.1 原则
- 眼睛是肖像的焦点，需要特别关注
- 提亮眼睛但不改变眼睛的颜色和形状
- 增强眼神光（catchlight）增加生动感

### 4.2 操作方法

**Lightroom方案：**
```
步骤：
1. 创建径向滤镜（Radial Filter），覆盖眼部区域
2. 调整参数：
   - Exposure: +0.3 ~ +0.5
   - Clarity: +15 ~ +25
   - Sharpness: +20 ~ +30
   - Dehaze: +5 ~ +10
3. 羽化边缘确保自然过渡
4. 确保不影响眼皮和眼眶阴影
```

**Photoshop方案：**
```
步骤：
1. 创建曲线调整图层（提亮）
2. 用黑色蒙版遮盖全部
3. 用白色柔边画笔（低不透明度3-5%）涂抹虹膜区域
4. 创建色相/饱和度调整图层
5. 微调虹膜饱和度（可选增强）
6. 新建图层，用白色画笔（极低不透明度1-2%）在瞳孔反光位置点涂
   → 增强眼神光（catchlight）
```

**醒图方案：**
```
路径：调整 → 人像 → 眼睛
- 眼睛提亮：20-40
- 眼睛放大：5-15（不要过大）
- 眼神光：20-40
```

### 4.3 注意事项
- 不要过度提亮导致眼睛失去立体感
- 不要改变瞳孔颜色
- 眼白提亮要适度，避免"发光"效果
- 两只眼睛的处理要对称
