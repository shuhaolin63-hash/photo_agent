# 人像调色

## 一、肤色校正

### 1.1 肤色标准
- 健康肤色应偏暖，有适当的红/黄/橙色调
- 亚洲人肤色参考值（HSL）：H: 15-30, S: 20-40, L: 55-75
- 白种人肤色参考值：H: 20-35, S: 15-30, L: 65-85
- 检测肤色问题的方法：看额头/鼻子两侧是否有不自然的颜色偏移

### 1.2 Lightroom肤色校正流程
```
步骤1：白平衡校正
- 取样器点击灰色区域（非皮肤）校正整体白平衡
- 或手动调整：色温5000-6500K，色调+5~+15

步骤2：HSL面板肤色调整
- Orange Hue: -5 ~ -10（校正肤色偏色）
- Orange Saturation: -5 ~ +10（根据需要调整饱和度）
- Orange Luminance: +5 ~ +20（提亮肤色）
- Red Saturation: -5 ~ -10（减少红色通道过饱和）
- Yellow Saturation: -5（减少黄色通道）

步骤3：校准面板
- Blue Primary Hue: 向左偏移 → 暖化肤色
- Red Primary Hue: 微调 → 校正肤色偏红/偏黄
- Green Primary Saturation: 微调 → 控制中间调饱和度

步骤4：蒙版局部校正
- 用颜色范围蒙版选中皮肤
- 对皮肤做独立的亮度/饱和度微调
```

### 1.3 Photoshop肤色校正流程
```
步骤1：曲线+灰场校正
- 创建曲线调整图层
- 在阴影/中间调/高光处设置灰场取样点
- 用灰场吸管点击照片中应为灰色的区域

步骤2：可选颜色
- 创建可选颜色调整图层
- 选择Red通道，调整Cyan/Black
- 选择Yellow通道，调整Magenta/Black
- 微调消除不自然的肤色偏色

步骤3：色相/饱和度
- 创建色相/饱和度调整图层
- 选择Red/Yellow/Orange通道
- 微调色相和饱和度
```

## 二、情绪色调

### 2.1 暖色调（温馨/复古/怀旧）
```
Lightroom方案：
- 白平衡：色温+10~+30，色调+5~+15
- 分离色调：高光 加暖（橙/黄色 H:30 S:10-20），阴影 加暖（橙/棕）
- 曲线：RGB微微S形（增强对比），R曲线高光上提（暖化）
- HSL：整体饱和度微降，保留暖色

效果：画面温暖柔和，适合情侣/家庭/童年主题
```

### 2.2 冷色调（清冷/忧郁/高级感）
```
Lightroom方案：
- 白平衡：色温-10~-30，色调-5~-15
- 分离色调：高光 加冷（蓝/青色 H:210 S:10-15），阴影 加冷（蓝/紫）
- 曲线：R曲线蓝色上提（增加蓝通道），B曲线压暗
- 色调曲线高光偏蓝，阴影偏青

效果：画面清冷高级，适合时尚/商务/个性人像
```

### 2.3 胶片色调（Vivid/柔和）
```
Lightroom方案：
- 颗粒：Amount 20-40, Size 25, Roughness 50
- 色彩偏差（色彩偏移）：阴影偏青（Teal），高光偏橙（Orange）
- 曲线：高光和阴影同时压暗（S形曲线+肩部/趾部压缩）
- HSL：整体饱和度+5~+15，冷暖对比增强

效果：现代胶片感，类似Kodak Portra 400效果
```

### 2.4 日系色调（通透/清新/淡雅）
```
Lightroom方案：
- 曝光：+0.3~+0.5（整体提亮）
- 对比度：-10~-20（降低对比度）
- 高光：-20~-40（压回高光）
- 阴影：+40~+60（提亮阴影）
- 饱和度：-5~-15（降低饱和度）
- 色调：+10~+20（偏粉）
- 清晰度：-5~-15（柔化整体）
- 分离色调：高光 浅粉色，阴影 浅蓝色

效果：清新淡雅，低对比度，偏粉的日系感
```

### 2.5 黑白人像调色
```
Lightroom方案：
- 转换为黑白（B&W面板）
- 调整各颜色通道的灰度值：
  - Red: +20~+40（皮肤偏亮）
  - Orange: +10~+30（肤色控制）
  - Yellow: +10~+20
  - Green: -20~-40（背景压暗）
  - Blue: -30~-50（天空/阴影压暗）
- 分离色调（可选）：
  - 高光：银色/暖白色
  - 阴影：深褐色/黑色

效果：经典黑白肖像，皮肤层次丰富，背景干净
```

## 三、胶片模拟

### 3.1 Kodak Portra 400 模拟
```
特征：柔和色彩、暖色调、细腻颗粒、低对比度、优秀肤色还原

Lightroom参数：
- 白平衡：色温+10~+15，色调+5~+10
- 曝光：+0.2
- 对比度：-15~-20
- 高光：-20
- 阴影：+30
- 饱和度：-5
- 清晰度：-5
- 曲线：轻微S形，肩部柔和压下
- HSL Orange Luminance: +10
- 分离色调：高光(暖白), 阴影(浅褐)
- 颗粒：Amount 15, Size 25, Roughness 40
```

### 3.2 Kodachrome 64 模拟
```
特征：高饱和度、暖色调、强烈对比、深沉阴影

Lightroom参数：
- 白平衡：色温+15~+20，色调+10
- 对比度：+20~+30
- 饱和度：+20~+30
- 鲜艳度：+10~+15
- 清晰度：+10
- 曲线：强S形，高光不过曝，阴影深沉
- HSL：Red+20, Yellow+15, Blue+10
- 分离色调：高光(金橙), 阴影(深蓝)
- 颗粒：Amount 25, Size 35, Roughness 50
```

### 3.3 Ilford HP5 Plus 模拟
```
特征：中等对比度、细腻颗粒、经典黑白

Lightroom参数：
- 转为黑白
- 对比度：+10~+15
- 高光：-10
- 阴影：-10
- 清晰度：+5
- 曲线：中等S形
- B&W面板：根据场景调整
- 颗粒：Amount 30, Size 30, Roughness 45
- 分离色调：高光(微银), 阴影(微褐)
```

### 3.4 Fuji Velvia 50 模拟
```
特征：极高饱和度、偏冷色调、高对比度、绿色和蓝色尤其浓烈

Lightroom参数：
- 白平衡：色温-5~0，色调-5
- 对比度：+25~+35
- 饱和度：+30~+40
- 鲜艳度：+15~+20
- 清晰度：+10
- 曲线：强S形
- HSL：Green+30, Blue+20, Red+10
- 颗粒：Amount 10, Size 20, Roughness 40
```

### 3.5 柯达Tri-X 400 模拟
```
特征：高对比度黑白、明显颗粒、深沉阴影

Lightroom参数：
- 转为黑白
- 对比度：+25~+35
- 阴影：-30~-50
- 清晰度：+10~+15
- 曲线：强S形，阴影压低
- 颗粒：Amount 40, Size 40, Roughness 55
- 分离色调：阴影(微褐) 增强胶片感
```
