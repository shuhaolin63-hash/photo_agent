# 风光基础修图

## 一、水平线校正

### 1.1 重要性
- 水平线倾斜是风光摄影中最常见也最容易修正的问题
- 即使1-2度的倾斜也会让观者感到"不对劲"
- 校正水平线是修图的第一步，也是最重要的一步

### 1.2 Lightroom校正方法
```
方法一：自动校正
- 工具栏 → 裁剪工具 → 自动（Auto）
- Lightroom会自动检测水平线并校正
- 注意：自动检测不一定准确，需要人工确认

方法二：水平仪工具
- 工具栏 → 裁剪工具 → 水平仪（Level工具）
- 在画面中沿已知水平的线段（地平线、水面、屋顶线）拖动
- Lightroom自动旋转校正

方法三：手动旋转
- 裁剪工具 → 旋转滑块
- 微调0.1-0.5度

方法四：Upright垂直校正
- Transform面板 → Upright Auto/Level/Vertical/Full
- 适用于建筑/城市风光，校正透视变形
- Level：仅校正水平
- Vertical：校正垂直透视
- Full：同时校正水平和垂直

参数参考：
- 水平旋转：通常在±3度以内
- 垂直缩放：透视校正后通常需要+5~+20的缩放补偿裁切损失
```

### 1.3 醒图方案
```
路径：剪裁 → 旋转
- 使用自动水平检测
- 或手动微调角度
- 透视校正：调整 → 透视校正
```

## 二、HDR（高动态范围）

### 2.1 HDR适用场景
- 高光比场景：日出日落、逆光、室内窗边
- 目标：保留高光细节 + 保留暗部细节，超越单张RAW的动态范围

### 2.2 拍摄HDR的基础要求
```
前期拍摄建议：
1. 使用三脚架
2. AEB包围曝光：至少3张（-2EV, 0, +2EV）
3. 5-7张包围曝光效果更好
4. 保持光圈不变，只改变快门速度
5. 使用低ISO减少噪点

注意：如果单张RAW的动态范围已经足够（如低光比场景），
不需要HDR，单张处理即可。
```

### 2.3 Lightroom HDR合成
```
方法一：HDR全景（Photo Merge → HDR）
- 选中包围曝光系列
- 右键 → Photo Merge → HDR
- 勾选 "Auto Align"（自动对齐）
- 勾选 "Auto Settings"（自动应用设置）
- 合并生成DNG文件

方法二：单张RAW的"伪HDR"（推荐日常使用）
- 高光滑块：-50 ~ -80（压回高光）
- 阴影滑块：+50 ~ +80（提亮阴影）
- 白色滑块：-20 ~ -40
- 黑色滑块：+10 ~ +30
- 清晰度：+10 ~ +20
- Dehaze：+5 ~ +15

这是大多数风光照片的最佳方案，避免了多张合成的复杂性。
```

### 2.4 HDR常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| **画面像油画** | HDR过度，清晰度过高 | 降低Dehaze和清晰度，降低对比度 |
| **光晕/伪影** | 对齐失败或运动物体 | 使用"去鬼影"功能，或用单张处理 |
| **色彩不自然** | 色调映射过度 | 降低饱和度，参考自然色彩 |
| **噪点增多** | 阴影提亮过度导致 | 适当降噪 + 不要过度提亮 |

## 三、对比度调整

### 3.1 对比度是风光照片的关键
- 对比度决定了画面的"立体感"和"层次感"
- 过低：画面扁平、灰蒙
- 过高：细节丢失、不自然
- 目标：自然、有层次、有呼吸感

### 3.2 Lightroom对比度调整方案
```
方法一：基础面板
- Contrast（对比度）滑块：±10~±30
- 仅调整中间调对比度，不影响高光和阴影

方法二：曲线（Tone Curve）——精确控制
- S形曲线：同时增强高光对比和阴影对比
  - 高光点：下拉10-15%
  - 暗部点：上提10-15%
- 单独调整RGB通道曲线：
  - R曲线提升 → 暖色调
  - B曲线提升 → 冷色调

方法三：高光/阴影面板
- Highlights（高光）：-20~-60（压回天空过曝）
- Shadows（阴影）：+30~+70（提亮前景暗部）
- Whites（白色）：-10~-30
- Blacks（黑色）：+10~+30

方法四：清晰度（Clarity）
- Clarity: +10~+30（增强中对比度，提升质感）
- 注意：人像慎用（会使皮肤粗糙）
- Dehaze: +5~+15（去雾，增强远景层次）
```

### 3.3 对比度调整策略

```
低光比场景（阴天、雾天）：
- Contrast: +15~+25
- S形曲线轻微
- Clarity: +10~+15

中等光比（晴天正面光）：
- Contrast: +5~+15
- S形曲线中等
- Clarity: +15~+20

高光比场景（日出日落、逆光）：
- Contrast: -5~+5
- 不使用强S形曲线
- Highlights: -50~-80
- Shadows: +50~+80
- 用高光/阴影面板替代全局对比度

```

## 四、其他基础调整

### 4.1 清晰度与锐化
```
Lightroom清晰度（Clarity）：
- 风光照片：+10~+30（提升岩石、云层纹理）
- 注意不要过度（>40），否则会出现"HDR油画"效果

Lightroom锐化（Sharpening）：
- Amount: 40~60
- Radius: 1.0（风光）
- Detail: 25~40
- Masking: 30~50（使用Alt键查看蒙版）

输出锐化（导出时）：
- 输出锐化：屏幕/哑光纸/光泽纸
- 锐化量：标准/高
```

### 4.2 降噪
```
Lightroom降噪面板：
- Luminance（亮度噪点）：
  - ISO 100-400：10~20
  - ISO 400-1600：20~40
  - ISO 1600-6400：40~70
  - ISO >6400：70~100
- Detail: 50
- Contrast: 0~20（保留对比度）
- Color（彩色噪点）：
  - 通常25~50即可
  - Detail: 50
```

### 4.3 镜头校正
```
Transform面板：
- 启用配置文件校正
- 选择对应镜头
- 移除色差（Remove Chromatic Aberration）：勾选
- 启用镜头校正（Enable Lens Corrections）：勾选
```
