# 风光调色

## 一、天空渐变

### 1.1 重要性
- 风光照片中天空通常占据很大面积，天空的色彩直接影响整体氛围
- 原始照片的天空往往平淡无奇，需要通过后期增强
- 天空与地面的色温通常不一致，需要分别调整

### 1.2 Lightroom天空渐变方案

**方法一：渐变滤镜（Graduated Filter）**
```
步骤：
1. 选择渐变滤镜工具
2. 从天空顶部向下拖动渐变线
3. 渐变线位置：地平线附近
4. 调整参数：
   - 色温：-20~-40（偏蓝冷，增强天空感）
   - 色调：-5~-15（偏青色）
   - 曝光：-0.3~-0.5（压暗天空，让云层更有层次）
   - 清晰度：+10~+20（增强云层纹理）
   - 饱和度：+10~+20
5. 羽化：50~80（确保自然过渡）
```

**方法二：线性渐变蒙版（新版LR）**
```
步骤：
1. 创建蒙版 → 选择"线性渐变"
2. 从天空顶部向下拖动
3. 精确控制覆盖范围
4. 应用色调和影调调整
```

**方法三：天空蒙版（AI检测）**
```
步骤（Lightroom 2023+）：
1. 创建蒙版 → 选择"天空"
2. Lightroom会自动检测天空区域
3. 微调蒙版边缘
4. 对天空区域单独应用调整
```

### 1.3 日出/日落天空增强
```
参数：
- 色温：+15~+30（暖化天空）
- 色调：+5~+15（偏橙）
- 曝光：-0.2~-0.4
- 饱和度：+15~+30
- 鲜艳度：+10~+20
- 分离色调高光：添加暖橙/金色（H:30-40, S:15-25）
```

### 1.4 阴天/蓝色天空增强
```
参数：
- 色温：-20~-40（偏蓝冷）
- Dehaze：+10~+20（去雾增强蓝天）
- 饱和度：+10~+20
- 蓝色通道饱和度：+15~+25
- 分离色调高光：添加浅蓝/青色（H:200-220, S:10-15）
```

## 二、白平衡创意

### 2.1 白平衡不仅是"校正"，更是"创作"

常规白平衡目的是让白色看起来是白色。但在风光摄影中，创意白平衡是营造氛围的重要手段。

### 2.2 暖白平衡（日落/金色时刻）
```
场景：日落/日出/沙漠/秋色
参数：
- 色温：6500-8000K（偏暖）
- 色调：+10~+25

效果：
- 天空更金黄
- 岩石/土地更温暖
- 整体氛围浪漫温暖
- 类似Charlie Waite的青蓝-金色对比
```

### 2.3 冷白平衡（冬季/清晨/阴天）
```
场景：雪山/冬日/清晨/雾气
参数：
- 色温：4000-5000K（偏冷）
- 色调：-10~-20

效果：
- 画面清冷纯净
- 蓝色/青色增强
- 雪景更冷冽
- 类似Charlie Waite的晨雾风格
```

### 2.4 色彩偏移（高级技巧）
```
Lightroom色彩偏移面板（Color Grading）：
- 阴影色调偏移：向Teal（青色）偏移
- 高光色调偏移：向Orange（橙色）偏移
- 这是经典的"Teal & Orange"电影色调

参数参考：
- 阴影：H:180-200, S:10-20
- 中间调：H:30-40, S:5-10
- 高光：H:25-35, S:15-25

效果：
- 画面具有电影感
- 暗部偏青，亮部偏橙
- 适用于沙漠/峡谷/海景
```

### 2.5 紫色调（薰衣草/紫色氛围）
```
参数：
- 色调：-10~-20（偏紫）
- 分离色调：高光 添加淡紫色
- 色彩偏移 中间调：H:280-300, S:10-15

适用场景：
- 夕阳后的天空
- 薰衣草田
- 梦幻/浪漫氛围
```

## 三、局部调色

### 3.1 前景独立调色
```
场景：前景是草地/花朵/岩石，与天空色彩/光线不同

Lightroom方案：
1. 创建蒙版 → 选择"物体"或"画笔"
2. 选中前景区域
3. 独立调整前景的：
   - 色温（通常前景偏暖，需要校正）
   - 曝光（通常前景偏暗，需要提亮）
   - 饱和度（增强前景色彩）
   - 清晰度（增强前景纹理）
4. 羽化蒙版边缘确保自然过渡
```

### 3.2 主体独立调色
```
场景：主体（如一棵特别的树/一座建筑）需要突出的色彩

Lightroom方案：
1. 创建径向滤镜或物体蒙版，选中主体
2. 调整参数：
   - 曝光：+0.3~+0.5（提亮主体）
   - 饱和度：+10~+20（增强主体色彩）
   - 清晰度：+10~+20（增强主体质感）
   - 色温/色调：独立调整色温
3. 对背景做反向调整（压暗/降低饱和度）形成对比
```

### 3.3 光线路径增强
```
场景：画面中有明显的光线路径（如穿透云层的光束）

Lightroom方案：
1. 创建径向滤镜，覆盖光束区域
2. 调整参数：
   - 曝光：+0.3~+0.7（提亮光束）
   - 色温：+10~+20（暖化光线）
   - 饱和度：+10~+15
3. 羽化边缘确保光束自然扩散

或使用画笔蒙版精确绘制光束形状
```

### 3.4 水面反射增强
```
Lightroom方案：
1. 创建线性渐变蒙版，覆盖水面区域
2. 调整参数：
   - 清晰度：+10~+20（增强水面纹理）
   - 饱和度：+5~+10
   - 对比度：+5~+10
3. 如果水面倒影天空色彩，可以独立调整水面色温

醒图方案：
- 滤镜 → 渐变，覆盖水面区域
- 调整 → 色调/光感
```

### 3.5 前景提亮/压暗（突出层次）
```
Lightroom方案：
1. 创建画笔蒙版，选中前景
2. 提亮前景：
   - Exposure: +0.3~+0.5
   - Shadows: +20~+40
3. 压暗前景（如果前景过亮）：
   - Exposure: -0.3~-0.5
   - Highlights: -30~-50

目标：让前景与中景、远景形成三个清晰的亮度层次
这是安塞尔·亚当斯风格风光照片的核心技法
```

## 四、常见风光调色问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| **天空色彩不自然** | 渐变滤镜边缘过硬 | 增加羽化（50-80），使用线性渐变替代硬边 |
| **整体偏色** | 白平衡调整过度 | 回调色温/色调至合理范围 |
| **前景/天空色温不一致** | 没有分别调整 | 使用蒙版分别调整前景和天空的色温 |
| **色彩过饱和** | Vibrance/Saturation过高 | HSL单独调整过饱和的通道 |
| **画面灰蒙** | Dehaze不足/对比度低 | Dehaze +10~+20，Clarity +10~+20 |
| **暗部色彩脏** | 阴影提亮后出现彩色噪点 | 阴影调色+降噪或使用黑白阴影 |
