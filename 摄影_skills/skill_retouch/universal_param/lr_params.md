# Lightroom通用参数模板

## 一、模板使用说明
以下参数模板基于Adobe Lightroom Classic CC，适用于RAW文件处理。参数值为基础参考值，应根据每张照片的具体情况进行微调。

## 二、人像模板

### 2.1 自然光人像（室外/窗边）
```
【基础面板】
- 曝光：+0.2 ~ +0.5
- 对比度：+5 ~ +10
- 高光：-15 ~ -30
- 阴影：+20 ~ +40
- 白色：-5 ~ -10
- 黑色：+5 ~ +10
- 清晰度：+5（保留皮肤质感）
- 鲜艳度：+5 ~ +10
- 饱和度：0

【色温色调】
- 色温：5200K ~ 5800K（根据实际光色调整）
- 色调：+5 ~ +10（偏暖肤色）

【HSL面板】
- Orange Luminance: +10 ~ +15（提亮肤色）
- Orange Saturation: 0 ~ +5
- Red Saturation: -5 ~ -10
- Yellow Saturation: -5

【锐化】
- Amount: 40
- Radius: 1.0
- Detail: 25
- Masking: 30

【颗粒】
- Amount: 10 ~ 20（可选，增加质感）
- Size: 25
- Roughness: 40

【暗角】
- Amount: -10 ~ -15（轻微暗角聚焦人像）
- 中点：50
- 羽化：50
```

### 2.2 室内人像
```
【基础面板】
- 曝光：+0.3 ~ +0.7（室内通常偏暗）
- 对比度：+10 ~ +15
- 高光：-20 ~ -40
- 阴影：+30 ~ +50
- 清晰度：+5

【色温色调】
- 色温：根据光源类型：
  - 钨丝灯：3500K ~ 4200K
  - 荧光灯：4000K ~ 5000K
  - LED灯：4500K ~ 5500K
  - 窗户自然光：5000K ~ 6500K

【色调曲线】
- 微S形曲线增强对比
- R通道：高光微提（暖化肤色）
```

## 三、风光模板

### 3.1 日出/日落风光
```
【基础面板】
- 曝光：+0.1 ~ +0.3
- 对比度：+10 ~ +20
- 高光：-40 ~ -70（保留天空层次）
- 阴影：+40 ~ +70（提亮前景）
- 白色：-15 ~ -30
- 黑色：+10 ~ +20
- 清晰度：+15 ~ +25
- 鲜艳度：+10 ~ +20
- 饱和度：+5

【色温色调】
- 色温：+10 ~ +25（暖化）
- 色调：+5 ~ +15

【HSL面板】
- Red Saturation: +10 ~ +20
- Orange Saturation: +10 ~ +15
- Yellow Saturation: +10
- Blue Saturation: +15 ~ +25
- Purple Saturation: +10

【色调曲线】
- S形曲线：高光下拉10-15%，暗部上提10-15%
- R曲线：高光微提5-10%（暖化高光）

【分离色调】
- 高光：H:30-40, S:10-20（金橙色）
- 阴影：H:200-220, S:5-10（浅蓝色）
- 平衡：偏暗部

【渐变滤镜-天空】
- 从天空向下拖渐变
- 色温：-15
- 曝光：-0.3
- 清晰度：+10

【暗角】
- Amount: -20 ~ -30
- 样式：优先高光
```

### 3.2 蓝天白云风光
```
【基础面板】
- 曝光：+0.2
- 对比度：+15 ~ +25
- 高光：-20 ~ -40
- 阴影：+30 ~ +50
- 清晰度：+15 ~ +25
- 鲜艳度：+15 ~ +25
- 饱和度：+5 ~ +10

【色温色调】
- 色温：-10 ~ -20（偏蓝冷天空）
- Dehaze: +10 ~ +20

【HSL面板】
- Blue Luminance: -10 ~ -20（压暗蓝天，让白云更突出）
- Blue Saturation: +15 ~ +25
- Blue Hue: -5 ~ -10

【渐变滤镜-天空】
- 色温：-20 ~ -30
- Dehaze: +10 ~ +15
```

### 3.3 阴天/雾天风光
```
【基础面板】
- 曝光：+0.3 ~ +0.5
- 对比度：-5 ~ +5（低对比度场景）
- 高光：-30 ~ -50
- 阴影：+50 ~ +70
- 白色：-20
- 黑色：+20
- 清晰度：+10 ~ +15
- Dehaze: +5 ~ +10

【色温色调】
- 色温：0 ~ +10（略暖化避免灰冷）
- 色调：0 ~ +5

【色调曲线】
- 减弱S形或使用平缓曲线
- 目标：柔和的层次感
```

## 四、星空模板

### 4.1 银河星空
```
【基础面板】
- 曝光：+0.5 ~ +1.5
- 对比度：+15 ~ +25
- 高光：-20 ~ -40
- 阴影：+60 ~ +80
- 清晰度：+20 ~ +35
- 鲜艳度：+10 ~ +20
- 饱和度：+5 ~ +10

【色温色调】
- 色温：-10 ~ -25（天空偏蓝）
- 色调：0 ~ +5

【HSL面板】
- Blue Saturation: +15 ~ +25
- Blue Luminance: +5 ~ +10
- Purple Saturation: +10 ~ +15
- Orange Saturation: +5 ~ +10
- Yellow Saturation: +5 ~ +10

【分离色调】
- 高光：H:210-240, S:8-12（淡蓝/淡紫）
- 阴影：H:210, S:5-8（深蓝）

【锐化】
- Amount: 50 ~ 70
- Radius: 1.0 ~ 1.5
- Detail: 20 ~ 30
- Masking: 40 ~ 60

【降噪】
- Luminance: 根据ISO调整（参考starry_process.md）
```

## 五、街拍/纪实模板

### 5.1 黑白街拍
```
【转为黑白】
- B&W面板或V+S=0

【基础面板】
- 曝光：根据场景
- 对比度：+20 ~ +35
- 高光：-15 ~ -30
- 阴影：+20 ~ +40
- 清晰度：+10 ~ +20

【B&W混色器】
- Red: +20 ~ +40（肤色变亮）
- Orange: +10 ~ +20
- Yellow: +5 ~ +10
- Green: -10 ~ -30
- Blue: -30 ~ -50
- Purple: -20 ~ -40

【色调曲线】
- S形曲线增强黑白反差
- 高光下拉15%，暗部上提15%

【颗粒】
- Amount: 30 ~ 50（Tri-X风格）
- Size: 35
- Roughness: 50

【暗角】
- Amount: -20 ~ -35
```

### 5.2 彩色街拍
```
【基础面板】
- 曝光：根据场景
- 对比度：+10 ~ +20
- 清晰度：+5 ~ +15
- 鲜艳度：+5 ~ +10

【HSL面板】
- 根据场景主色调调整
- 通常：Red +10, Blue +10

【颗粒】
- Amount: 10 ~ 20（轻微颗粒增加纪实感）
```

## 七、黑白风光模板

```
【转为黑白】

【基础面板】
- 对比度：+25 ~ +40
- 高光：-20 ~ -40
- 阴影：+20 ~ +40
- 清晰度：+15 ~ +25

【B&W混色器】（根据场景调整）
- 红滤镜效果：Red +40（天空变暗，白云突出）
- 橙滤镜效果：Orange +25
- 黄滤镜效果：Yellow +15（中等对比增强）
- 绿滤镜效果：Green -20（压暗植被）
- 蓝滤镜效果：Blue -50（天空极暗）

【色调曲线】
- 强S形曲线
- 高光下拉15-20%
- 暗部上提10-15%
- 肩部和趾部适度压缩

【分离色调】
- 高光：H:35, S:5（暖银色）
- 阴影：H:210, S:5（微蓝）

【颗粒】
- Amount: 20 ~ 35
- Size: 25 ~ 35
- Roughness: 45 ~ 55

【暗角】
- Amount: -25 ~ -40
```
