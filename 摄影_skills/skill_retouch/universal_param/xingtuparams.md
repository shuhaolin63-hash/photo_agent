# 醒图（XingTu）参数模板大全

> 适用平台：醒图APP（字节跳动旗下移动端修图工具）
> 版本参考：醒图 v4.x（2024-2025持续更新版本）
> 说明：以下所有参数均基于醒图真实滑块范围，推荐值为实际修图常用区间，非机器生成泛化数值。

---

## 一、基础调整（人像通用）

醒图的"调整"面板包含以下核心参数，路径为：底部工具栏 → 调整。

### 1.1 光感（Light）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +5 ~ +25（人像）；+10 ~ +30（风光） |
| 效果说明 | 醒图独有参数，近似于"智能曝光"。不同于单纯的亮度提升，光感会基于AI分析画面内容进行局部自适应提亮，对人脸区域有优先提亮倾向。拉高后画面整体变亮但不易过曝，适合快速挽救欠曝照片。负值压暗画面。与人像模式联动时效果更自然。 |

### 1.2 亮度（Brightness）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +3 ~ +15（人像）；+5 ~ +20（风光） |
| 效果说明 | 全局均匀提亮/压暗，不区分高光阴影。效果类似屏幕背光调节，对所有像素等量偏移。与"光感"的区别：亮度是线性均匀调整，光感有局部智能加权。人像修图中亮度一般微调即可，主要依赖光感和曝光。 |

### 1.3 曝光（Exposure）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +5 ~ +20（人像）；+10 ~ +35（风光暗光） |
| 效果说明 | 模拟相机曝光补偿，改变整体曝光量。与亮度不同，曝光按照EV（曝光值）的非线性曲线调整，高光区域变化更显著。拉高时亮部先溢出，暗部后溢出；降低时暗部先压死。是曝光校正的首选参数。 |

### 1.4 对比度（Contrast）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +5 ~ +15（人像）；+10 ~ +25（风光） |
| 效果说明 | 同时拉伸亮部和暗部，中间灰点不动。正方向让亮的更亮、暗的更暗，画面更有"力度"；负方向让明暗趋近，画面偏"灰平"。人像修图中对比度不宜过高（>+20会显得生硬），风光可适当加大增强层次。 |

### 1.5 高光（Highlights）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | -15 ~ -35（恢复过曝）；-5 ~ -10（日常微调） |
| 效果说明 | 仅影响画面中较亮的部分（约上1/3色调范围）。负值压回过曝区域，是拯救天空、窗户反光、皮肤高光的首选参数。正值让亮部更亮（少用）。人像修图中高光通常需要压低-10~-30来恢复皮肤和衣服的亮部细节。风光中用于控制天空不过曝。 |

### 1.6 阴影（Shadows）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +15 ~ +35（提亮暗部）；+20 ~ +40（逆光人像） |
| 效果说明 | 仅影响画面中较暗的部分（约下1/3色调范围）。正值提亮阴影区域，是逆光人像"提亮面部"的核心手段。负值加深阴影增加戏剧感。风光摄影中提亮阴影可以展现暗部细节（树林、岩石纹理），但过度提亮（>+50）会引入噪点。 |

### 1.7 色温（Temperature）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +5 ~ +15（暖调人像）；-5 ~ -15（冷调风光）；+15 ~ +30（日落风光） |
| 效果说明 | 控制画面整体冷暖偏移。正值偏黄/橙（暖色），负值偏蓝（冷色）。人像修图中微微偏暖（+5~+15）可以让肤色更健康；钨丝灯/暖光灯拍摄的环境需要偏冷（-10~-20）校正白平衡。风光摄影中日落/黄金时段可强化暖色（+15~+30），蓝调时刻可强化冷色（-15~-30）。 |

### 1.8 色调（Tint）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +3 ~ +8（人像微调）；-5 ~ -10（偏绿校正） |
| 效果说明 | 控制品红-绿色的色偏。正值偏品红/洋红，负值偏绿。用于精细白平衡校正。荧光灯环境常偏绿（色调-5~-15校正），部分肤色偏黄时可加少量品红（+3~+8）让肤色更通透。人像修图中色调调整幅度通常很小。 |

### 1.9 饱和度（Saturation）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | 0 ~ +10（人像）；+10 ~ +25（风光）；-100（转黑白替代方案） |
| 效果说明 | 全局等量调整所有颜色的饱和度，包括已经很饱和的颜色。正值所有颜色都变得更浓烈，负值所有颜色都变淡。与自然饱和度的区别：饱和度是"一刀切"，自然饱和度有智能加权。人像修图中饱和度通常不大幅调整，避免肤色失真；风光可适当提升。拉到-100即为去色/黑白。 |

### 1.10 自然饱和度（Vibrance）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +8 ~ +20（人像）；+15 ~ +30（风光） |
| 效果说明 | 智能饱和度调整。会优先提升画面中低饱和度的颜色（如灰蓝的天空、泛黄的草地），对已经高饱和的颜色提升较小，并且对肤色有保护机制（不会让肤色过度饱和变橙）。人像修图中更推荐用自然饱和度替代饱和度，+10~+20可以让画面更鲜活但不影响肤色真实感。 |

### 1.11 锐化（Sharpen）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 15 ~ 30（人像）；25 ~ 45（风光）；30 ~ 50（细节纹理） |
| 效果说明 | 增强图像边缘的对比度，让细节看起来更清晰。醒图的锐化算法较温和，适合手机照片。人像中适度锐化（15~30）可以让眼睛、头发更有神，但过高的锐化（>50）会放大噪点并让皮肤显粗糙。风光中可以加到25~45增强建筑纹理、树叶细节。建议在导出前最后一步做锐化。 |

### 1.12 清晰度（Clarity）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +3 ~ +10（人像）；+10 ~ +25（风光）；-5 ~ -15（柔焦人像） |
| 效果说明 | 增强中间调的局部对比度，让画面更有"质感"和"立体感"。正方向增强纹理细节（皮肤毛孔、衣服面料、石头纹理会更明显）；负方向让画面偏柔和、有朦胧感。人像修图中清晰度要谨慎使用，>+15会暴露皮肤瑕疵（毛孔、皱纹），负值（-5~-15）可以模拟柔焦效果。风光中清晰度+10~+25可以大幅增强画面质感。 |

### 1.13 褪色（Fade）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 10 ~ 25（复古效果）；20 ~ 40（胶片模拟） |
| 效果说明 | 降低画面整体对比度并让暗部略微泛灰，模拟老照片/胶片的褪色效果。正方向让黑色不再纯黑（暗部泛灰），高光也略微柔化，整体感觉"旧旧的"。与降低对比度不同，褪色主要影响暗部的纯度，不会让中间调也变灰。复古/胶片风格的核心参数。 |

### 1.14 颗粒（Grain）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 15 ~ 30（胶片模拟）；8 ~ 15（轻微质感）；30 ~ 50（强颗粒艺术感） |
| 效果说明 | 叠加胶片颗粒噪点效果。醒图的颗粒算法模拟了真实胶片的银盐颗粒分布，比纯随机噪点更自然。人像修图中少量颗粒（10~20）可以掩盖手机摄影的数字噪点并增加"胶片感"；风光中一般不用。注意：颗粒会略微降低画面锐度，如果同时使用锐化建议先锐化后加颗粒。 |

### 1.15 暗角（Vignette）

| 属性 | 值 |
|------|-----|
| 滑块范围 | -100 ~ +100 |
| 默认值 | 0 |
| 推荐值区间 | +15 ~ +30（人像聚焦）；+20 ~ +40（电影感/戏剧感） |
| 效果说明 | 正方向让画面四角/边缘变暗，负方向让四角变亮。暗角是引导视觉焦点到画面中心的有效手段，人像和静物中常用。正值+15~+30产生自然暗角，+40以上暗角很明显。醒图的暗角算法是椭圆形渐变，过渡比较自然。风光摄影中慎用，除非刻意营造氛围。 |

## 二、人像专属调整

醒图的"人像"面板提供AI驱动的人像美颜功能。路径：底部工具栏 → 人像（或"美颜"）。

### 2.1 美颜级别

| 属性 | 值 |
|------|-----|
| 调节方式 | 滑块或预设档位（通常为"自然/精细/高级"三档） |
| 推荐选择 | 自然档 / 总体强度30~50 |
| 效果说明 | 醒图美颜采用AI面部识别，自动检测五官位置后进行美化。建议选择"自然"档位，或手动将各项参数控制在合理范围内，避免"蛇精脸""塑料感"。核心原则：每项参数单独不超过30，整体效果才自然。 |

### 2.2 磨皮（Smoothing）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 25 ~ 40（日常自然）；40 ~ 55（社交平台）；10 ~ 20（保留质感） |
| 效果说明 | 柔化皮肤纹理，去除细小瑕疵和毛孔。醒图的磨皮算法支持"保留质感"模式（部分版本），在磨皮的同时尽量保留皮肤真实纹理。25~40为日常社交分享的甜点区间，既干净又不失真。超过60皮肤会出现明显的"塑料涂抹"痕迹。写真/商业人像建议控制在20~35。 |

### 2.3 瘦脸（Face Slim）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 5 ~ 15（微调自然）；15 ~ 25（明显效果）；0（男生/纪实人像） |
| 效果说明 | AI识别下颌线后向内收缩面部轮廓。醒图支持分区域调整（下颌、颧骨等），比全局瘦脸更精确。5~15的微调效果自然不易察觉，是社交媒体人像的常用范围。超过30会明显失真，下颌线会显得不自然。建议配合"面部立体"参数一起调整效果更好。 |

### 2.4 大眼（Eye Enlarge）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 5 ~ 15（自然微调）；15 ~ 25（明显效果）；0（男生/写实风格） |
| 效果说明 | 放大眼睛瞳孔区域。醒图AI会自动识别双眼位置并等比例放大。小幅调整（5~15）可以弥补广角镜头拍摄时面部边缘畸变导致的眼睛偏小问题。超过30眼睛会显得不自然，瞳孔比例失调。建议同步调整"亮眼"参数让眼睛更有神。 |

### 2.5 面部提亮（Face Brighten）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 15 ~ 30（日常）；30 ~ 45（逆光/暗光补偿） |
| 效果说明 | 智能识别面部区域并单独提亮，不影响背景和其他区域。非常适合逆光拍摄时面部偏暗的情况，或者室内光线不均时提亮面部阴影。与全局"光感""亮度"不同，面部提亮只作用于皮肤区域，不会让背景也过曝。15~30为自然提亮范围。 |

### 2.6 去黑眼圈（Dark Circle Removal）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 15 ~ 35（轻度黑眼圈）；35 ~ 55（中度）；55 ~ 70（重度） |
| 效果说明 | AI定位眼下区域进行提亮和色彩中和，消除黑眼圈和眼袋。15~30可以处理轻度疲劳痕迹，效果非常自然。注意不要过度使用，否则眼下区域会显得不自然的白亮，与周围皮肤形成明显色差。建议配合"面部提亮"一起使用，过渡更均匀。 |

### 2.7 牙齿美白（Teeth Whiten）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 20 ~ 40（自然白）；40 ~ 60（明显白） |
| 效果说明 | AI识别牙齿区域并提亮、去黄。20~40可以达到自然美白效果，牙齿看起来干净健康。超过60会显得不自然（荧光白），与面部肤色形成过大反差。微笑照/证件照中常用，日常随拍一般不需要。 |

### 2.8 祛痘（Blemish Removal）

| 属性 | 值 |
|------|-----|
| 滑块范围 | 0 ~ 100 |
| 默认值 | 0 |
| 推荐值区间 | 按需使用，通常30 ~ 60 |
| 效果说明 | AI自动检测面部痘印、斑点、瑕疵并进行修复。醒图采用"内容感知填充"算法，用周围皮肤纹理覆盖瑕疵区域。对于明显痘痘/疤痕建议单独使用"修复"工具手动点涂效果更好。祛痘滑块更适合处理大面积的细小瑕疵和肤色不均。 |

## 四、HSL分色调整

### 4.1 醒图HSL面板使用方法

路径：底部工具栏 → 调整 → 向下滑动找到"色相/饱和度/明度"（或"混色器"）。

醒图提供8个颜色通道，每个通道可独立调节三个维度：
- **色相（Hue）**：-100 ~ +100，左右移动该颜色在色轮上的位置。例如将红色色相-50可以将其偏移为橙色。
- **饱和度（Saturation）**：-100 ~ +100，增加或降低该颜色的浓度。降低到-100则该颜色完全去色。
- **明度（Luminance）**：-100 ~ +100，变亮或变暗该颜色。与整体亮度不同，仅影响选定颜色范围。

### 4.2 八通道调整思路

| 通道 | 影响范围 | 常用调整方向 |
|------|---------|-------------|
| 红色（Red） | 嘴唇、红色衣服、红花、交通标志 | 色相微调-5~-10让嘴唇更自然；饱和度+5~+10突出红色元素 |
| 橙色（Orange） | 肤色（主要）、秋天树叶、木纹 | **人像核心通道**，色相+3~+8让肤色偏暖黄；饱和度-5~-15降低肤色过饱和；明度+5~+10提亮肤色 |
| 黄色（Yellow） | 阳光、草地、柠檬、皮肤高光 | 色相-5~-10偏绿可以让草地更翠绿；饱和度+5~+10强化阳光感 |
| 绿色（Green） | 树叶、草地、蔬菜、绿色衣服 | 色相+5偏青绿更"日系"；饱和度+10~+20强化自然风光的绿色 |
| 青色（Cyan） | 晴空、水面、青色衣服 | 色相-5偏蓝让天空更蓝；饱和度+10~+15增强水天一色效果 |
| 蓝色（Blue） | 天空、大海、蓝色衣服、阴影 | 色相-5~-10让天空更深邃；饱和度+15~+25强化蓝天；明度-10~-20压暗天空增加戏剧感 |
| 紫色（Purple） | 薰衣草、晚霞、紫色衣服 | 饱和度+5~+10突出紫色元素；明度-5~-10增加紫色神秘感 |
| 品红（Magenta） | 粉色花朵、品红衣服、霓虹灯 | 饱和度+5~+10强化粉色；色相+5偏红让粉色更暖 |

## 五、局部调整

醒图的局部调整功能路径：底部工具栏 → 调整 → 局部（或"蒙版"）。

### 5.1 径向滤镜（Radial Filter）

**功能说明**：在画面上放置一个椭圆形选区，选区内部（或外部）应用调整参数。中心影响最大，向边缘渐变衰减至无影响。

**操作方法**：
1. 进入"局部调整"面板
2. 选择"径向"工具
3. 在目标区域点按并拖拽确定大小
4. 调整所选参数（光感、色温、饱和度等）
5. 拖拽选区边缘调整范围，拖拽中心点调整位置
6. 可添加多个径向滤镜叠加效果

**使用场景**：
- **人像面部提亮**：在面部放置径向滤镜，光感+15~+25，范围刚好覆盖面部
- **主体聚焦**：在主体上放置径向滤镜，清晰度+10，同时添加暗角+15让背景更暗
- **灯光效果模拟**：在光源方向放置径向滤镜，色温+20，光感+10，模拟暖光照射
- **背景虚化增强**：在背景区域反向使用径向滤镜，清晰度-10~-15，降低背景存在感

### 5.2 线性渐变（Linear Gradient）

**功能说明**：创建一条直线，从线条一侧向另一侧线性渐变应用调整参数。适合处理具有明显分界线的场景（天空/地面、前景/背景）。

**操作方法**：
1. 进入"局部调整"面板
2. 选择"线性"工具
3. 在画面上点按并拖拽出渐变方向线
4. 调整参数
5. 可旋转渐变方向、调整渐变范围
6. 可添加多条渐变线叠加

**使用场景**：
- **天空压暗/增强**：从画面顶部向下拖拽渐变，高光-20~-30，色温-5~-10，蓝色饱和度+15
- **地平线暖色叠加**：从地平线处向上拖拽，色温+15~+20，橙色饱和度+10（模拟日落暖光）
- **前景压暗**：从画面底部向上拖拽，光感-10~-15，阴影-10，让前景更暗衬托主体
- **水下/水面分界处理**：在水面交界处建立渐变，分别调整水面以上和以下的参数

### 5.3 蒙版笔刷（Mask Brush）

**功能说明**：手动涂抹绘制蒙版区域，精确控制调整范围。支持调整笔刷大小、硬度、不透明度，支持"涂抹"和"擦除"两种模式。

**操作方法**：
1. 添加一个全局调整层（如调整光感、色温等）
2. 点击该调整层进入蒙版模式
3. 选择"蒙版笔刷"
4. 用手指涂抹需要应用调整的区域
5. 涂抹错误可切换到"擦除"模式修正
6. 调整笔刷大小（双指缩放）和硬度

**使用场景**：
- **局部肤色校正**：全局降低橙色饱和度后，用蒙版笔刷擦除非皮肤区域，只保留肤色调整效果
- **局部提亮**：在阴影中的人物面部，用笔刷精确涂抹面部区域后提亮
- **天空替换辅助**：用笔刷精确涂抹天空区域，单独调整天空颜色
- **背景分离**：对背景单独应用冷色调调整，前景保持暖色，制造色彩对比
- **去除局部色偏**：荧光灯照射区域偏绿，用笔刷涂抹该区域后单独调整色调

### 5.4 局部调整实战示例

**示例：逆光人像面部提亮+天空压暗**
```

- 光感+10，对比度+8，高光-15，阴影+20

步骤2：添加径向滤镜（面部）
- 位置：覆盖面部区域
- 光感：+20
- 亮度：+10
- 面部提亮：+15

步骤3：添加线性渐变（天空）
- 方向：从画面顶部向下
- 范围：覆盖天空1/3区域
- 高光：-20
- 蓝色饱和度：+15
- 色温：-5

步骤4：如需要，用蒙版笔刷精细修正边缘溢出区域
```
