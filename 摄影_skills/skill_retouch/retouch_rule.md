# 修图总则

## 一、修图伦理

### 1.1 纪实摄影修图底线
- **允许**：曝光校正、白平衡调整、裁剪、对比度调整、色彩校正、锐化、降噪
- **不允许**：添加/删除元素、拼接/合成、改变事实性信息、改变人物外貌（新闻用途）
- **灰色地带**：局部提亮/压暗（允许但不夸张）、克隆工具去除杂物（谨慎使用）

### 1.2 艺术摄影修图自由度
- 艺术摄影允许更大幅度的后期处理
- 但仍应标注"后期合成"或"数字艺术"
- 尊重被摄者的基本形象（人像不恶意丑化或过度美化）

### 1.3 社交媒体修图准则
- 避免过度美颜导致的不自然效果
- 适度修图可以提升照片质量，过度修图会适得其反
- "三分拍七分修"不等于"改头换面"

## 二、修图流程总纲

### 2.1 标准修图流程（7步法）

```
步骤1：全局评估
  - 观察原图，识别技术问题和风格方向
  - 确定修图目标（修正技术问题 / 追求特定风格）

步骤2：基础校正
  - 水平线校正（风光）
  - 镜头畸变校正
  - 色差校正
  - 暗角校正

步骤3：曝光与影调
  - 曝光校正（高光/阴影/白色/黑色）
  - 对比度调整
  - 曲线调整（精确影调控制）

步骤4：色彩调整
  - 白平衡校正/创意白平衡
  - HSL面板（色相/饱和度/明度）
  - 分离色调（高光/阴影着色）

步骤5：局部精修
  - 局部提亮/压暗（遮罩+调整）
  - 局部色彩调整
  - 杂物去除（克隆/修复画笔）

步骤6：锐化与降噪
  - 降噪处理（先降噪）
  - 锐化处理（后锐化）
  - 输出锐化

步骤7：输出设置
  - 分辨率与格式
  - 色彩空间（sRGB/Adobe RGB/ProPhoto RGB）
  - 导出参数
```

## 三、软件适配指南

### 3.1 Adobe Lightroom Classic

**适用场景**：RAW文件处理、批量修图、影调与色彩的全局和局部调整

**核心优势**：
- 非破坏性编辑，所有调整可回退
- 强大的RAW处理引擎（Adobe Camera Raw）
- 高效的批量处理能力
- 渐变滤镜、径向滤镜的精确局部调整

**修图流程适配**：
- 步骤1-6全部在Lightroom中完成（大多数照片）
- 复杂的合成工作导出到Photoshop
- 人像精细修皮在Photoshop中完成

**推荐设置**：
- 导入预设：ProPhoto RGB色彩空间
- 预览模式：标准
- 锐化： amount 40, radius 1.0, detail 25, masking 50
- 降噪：luminance 根据ISO自动调整, detail 50

### 3.2 醒图（XingTu）

**适用场景**：手机端快速修图、社交媒体分享、创意滤镜效果

**核心优势**：
- 手机端最强大的修图工具之一
- 丰富的预设滤镜和调整工具
- 支持图层和蒙版操作
- 人像美颜功能（适度使用）

**修图流程适配**：
- 替代Lightroom进行手机端修图
- 社交媒体快速出图的首选工具
- 创意效果和滤镜的快速尝试

**对应Lightroom功能表**：
| Lightroom功能 | 醒图对应功能 |
|---------------|---------------|
| 基础面板（曝光/对比度等） | 调整 → 光感/亮度/对比度 |
| 白平衡 | 调整 → 色温/色调 |
| HSL面板 | 调整 → 色调分离/色相饱和度 |
| 渐变滤镜 | 滤镜 → 渐变 |
| 径向滤镜 | 滤镜 → 光效/聚光 |
| 裁剪 | 剪裁 |
| 克隆工具 | 修复 → 涂抹修复 |
| 曲线 | 曲线 |
| 分离色调 | 调整 → 色调/色温 |

### 3.3 Adobe Photoshop

**适用场景**：精修人像、合成图像、创意后期、精确像素级操作

**核心优势**：
- 图层系统支持复杂的非破坏性编辑
- 精确的选区和蒙版操作
- 专业级修图工具（仿制图章、修复画笔、内容识别填充）
- 频率分离磨皮等专业人像修图技术

**何时需要Photoshop**：
- 精修人像（磨皮、液化、眼睛提亮等）
- 复杂的杂物去除（多元素场景）
- 全景拼接
- HDR合成
- 数字艺术创作

## 四、修图原则总结

### 4.1 修图核心原则

1. **先全局后局部**：先做全局调整，再做局部精修
2. **先影调后色彩**：先确定明暗关系，再调整色彩
3. **先降噪后锐化**：先去除噪点，再进行锐化
4. **少即是多**：宁可不修，不可过度
5. **保持自然**：修图的最高境界是看不出修过的痕迹
6. **备份原图**：永远保留原始RAW文件

### 4.2 常见修图错误

| 错误 | 表现 | 正确做法 |
|------|------|----------|
| **过度磨皮** | 皮肤像塑料，无纹理 | 频率分离+适度磨皮，保留皮肤纹理 |
| **过度液化** | 身体比例失真 | 微调为主，保持自然比例 |
| **过饱和** | 色彩刺眼，不自然 | 适度饱和度+HSL精确控制 |
| **过度锐化** | 出现白色光晕和噪点 | 控制锐化量+使用遮罩 |
| **过度暗角** | 画面四周严重变暗 | 轻微暗角或不加暗角 |
| ** HDR过度** | 画面像油画，不自然 | 自然HDR+渐变滤镜替代 |
